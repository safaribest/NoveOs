## 二、6个选题建议

### 选题1 ★★★★★ | 无CP大女主 + 考据式穿越

**品类**: 穿越/古言（女频）
**一句话hook**: 历史系博士生穿成和亲公主，用专业考古知识在匈奴王庭生存、夺权、建立自己的政权——她没有爱上任何人
**目标读者**: 22-35岁女性，本科以上学历
**核心卖点**:
- 无CP（最热趋势），专注女主事业线
- 考据式穿越（2025十大关键词），知识密度=竞争壁垒
- 匈奴王庭背景（差异化场景，非套路后宫）


### 选题2 ★★★★☆ | 规则怪谈 + 人性博弈

**品类**: 悬疑（男频）
**一句话hook**: 一个普通程序员被拉入致命游戏，每条规则背后都是一个真实玩家的死亡——他必须从规则里还原出每个人的死法，才能找到活着出去的路
**目标读者**: 18-30岁男性，喜欢智力挑战
**核心卖点**:
- 《十日终焉》证明了规则怪谈的爆款潜力
- 人性博弈=智力爽感，不是打斗爽感
- 程序员主角=读者代入感强
- 每3-5章一个小反转，每卷一个大反转


### 选题3 ★★★★☆ | 修仙2.0 + 现代职场

**品类**: 都市修仙（男频）
**一句话hook**: 修仙界渡劫失败落入现代，发现大厂的OKR考核比渡劫还难——他用两千年的修炼经验，在互联网公司杀出一条修仙式的晋升之路
**目标读者**: 22-35岁职场男性
**核心卖点**:
- 修仙2.0=2025十大关键词
- "职场修仙"=修仙+打工人情绪双杀
- 幽默爽文风格，轻松不沉重
- 发疯文学元素自然嵌入


### 选题4 ★★★★☆ | 末世科幻 + 无CP女强

**品类**: 科幻末世（女频）
**一句话hook**: 全球植物一夜之间疯狂生长吞噬城市，一个植物学研究生成了人类最后的希望——她能听懂植物的语言，但植物要的不是和平
**目标读者**: 20-35岁女性
**核心卖点**:
- 末世=情绪释放出口
- 女性主角+专业能力=无CP大女主变体
- 植物学知识=考据式+行业细节
- 短剧改编潜力高


### 选题5 ★★★☆☆ | 两界穿梭 + 低武仙侠

**品类**: 玄幻（男频）
**一句话hook**: 一个现代地质勘探员发现地下存在灵气矿脉——两个世界开始为了资源互相试探、交易、背叛。他不是最强修士，却是唯一懂地质的人
**目标读者**: 18-30岁男性
**核心卖点**:
- 两界穿梭=蓝海赛道
- 地质知识=差异化竞争壁垒
- 低武=力量体系不崩
- 资源争夺=天然冲突引擎


### 选题6 ★★★☆☆ | 年代种田 + 发疯文学

**品类**: 年代重生（女频）
**一句话hook**: 996社畜穿成70年代知青，她不搞科研不考大学，凭借现代超市陈列逻辑和社群运营手段，在供销社时代建起了第一个"带货"帝国
**目标读者**: 25-40岁职场女性
**核心卖点**:
- 年代文=番茄算法最适配类型之一
- 发疯文学气质+种田文底子=有留存有传播
- 供销社→直播带货=古今映射笑点
- 适合短剧改编


## 三、写法操作公式

### 书名公式（番茄特色）
```
男频: {身份反差} + {金手指} + {结果暗示}
  例: 《开局一个系统，我在地府当公务员》《全球神祇：我能复制万物》

女频: {身份标签} + {冲突暗示} + {情感锚}
  例: 《和亲公主：我在匈奴建了一座城》《她不当贵妃以后，全天下来求她》
```

### 开头500字公式
```
具体事件(100字) → 冲突升级(200字) → 金手指/特质展示(100字) → 悬念(100字)
规则: 从事件的倒数第二个节点开始，不要从第一个
```

### 节奏公式（番茄算法适配）
```
每章: 试炼(1-2个) + 成长收获 + 新悬念
每3-5章: 小高潮（打脸/反转/揭秘）
每20章: 中高潮（卷收官+新卷预告）
每卷: 埋一个贯穿到尾的长线钩子
```

## 四、避坑红线

1. **不要追短期热词**——"发疯文学""NPC文学"是情绪标签不是品类，必须嫁接在扎实的品类骨架上
2. **不要在大品类做无差异化的标准品**——玄幻/都市/总裁必须先回答"我和同类有什么不同"
3. **不要忽视短剧改编潜力**——番茄6700+作品进入短剧改编，你的书从第一章就应该考虑"能拍吗"
4. **不要忽视女性读者的付费意愿**——番茄女用户占比55%，无CP大女主的完读率和付费远超甜宠
5. **不要开书太早**——大神共识：存稿5-8万字、完整大纲确认完毕、开头改到想吐，再发书

## 五、我的推荐：选题1

**《和亲公主：我在匈奴建了一座城》**

理由：
- 踩中番茄当前最大的增量风口（无CP大女主年增320%）
- 考据式穿越=知识壁垒=不是谁都能模仿
- 女性读者占比55%，女频盘子比男频大
- 无CP=天然规避"感情线写不好"的AI写作短板
- 短剧改编潜力极高（大女主+历史+逆袭=短剧热门元素三合一）
- 番茄编辑说："女频最缺的不是甜宠，是有脑子的女主"
