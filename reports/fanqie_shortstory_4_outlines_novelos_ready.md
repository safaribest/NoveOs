# 第一本：《毒杀重生后，我用医书屠了贵妃》

## 一、故事结构分析

### 三幕式结构

**第一幕：重生与布局（第1-4章）**
- 沈知微被毒杀，重生回三年前
- 接到贵妃传召，意识到危机提前
- 第一次用医术反击，建立“我不是任人宰割的棋子”人设
- 核心悬念：她能否改变命运？

**第二幕：权力博弈（第5-9章）**
- 卷入丽嫔小产、二皇子药碗等事件
- 发现二皇子身世秘密和三年前瘟疫真相
- 与贵妃、皇后、师父多方周旋
- 核心悬念：秘密如何在不暴露自己的情况下被皇帝知道？

**第三幕：复仇与超脱（第10-12章）**
- 假死脱身，公开指证贵妃
- 贵妃罪行败露，自饮毒酒
- 沈知微拒绝封赏，出宫开女子医馆
- 核心悬念：她能否真正获得自由？

### 核心悬念系统

| 悬念 | 埋设位置 | 揭晓位置 | 功能 |
|------|---------|---------|------|
| 沈知微为何被毒杀 | 第1章 | 第6章 | 驱动主线 |
| 三年前瘟疫真相 | 第3-4章 | 第6章 | 揭露贵妃动机 |
| 二皇子是否皇帝亲生 | 第5章 | 第6章 | 致命证据 |
| 贵妃能否被扳倒 | 第7章 | 第10章 | 终极冲突 |
| 沈知微能否离宫 | 第10章 | 第12章 | 结局升华 |

### 规则系统（世界观规则）

1. 后宫等级：皇后 > 贵妃 > 嫔妃 > 女官/医女
2. 太医署规则：医女可为后宫诊病，但不可介入宫斗
3. 毒药规则：宫中用毒常见，但需隐藏痕迹
4. 重生规则：主角保留前世记忆，但身体能力不变

## 三、写作硬规则

1. **开篇 500 字内**：必须出现主角死亡+重生+接到贵妃传召。
2. **每章字数**：1500-2200 字，适配番茄碎片化阅读。
3. **每章结构**：至少一个冲突单元 + 一个爽点/钩子。
4. **对话占比**：约 40%，用对话推动权力博弈。
5. **禁用词**：避免“然而/不得不说/众所周知/突然/竟然/原来/与此同时”。
6. **去 AI 味**：加入具体药材名、诊脉细节、宫廷礼仪，避免华丽空洞描写。
7. **情绪曲线**：压抑 → 反击 → 智斗 → 大释放，每 2-3 章一个小高峰。

# 第二本：《规则怪谈：我能看见死亡弹幕》

## 一、故事结构分析

### 三幕式结构

**第一幕：入局（第1-3章）**
- 陆恒被拉入机械厂副本
- 发现弹幕能力，但信息真假难辨
- 第一次玩家死亡，建立副本危险性
- 核心悬念：弹幕能不能信？

**第二幕：解谜（第4-8章）**
- 夜班任务、火灾档案、厂长诡异相继出现
- 陆恒用弹幕+逻辑推理逐步接近真相
- 李师傅守门人身份暴露
- 核心悬念：真正通关方式是什么？

**第三幕：破局（第9-12章）**
- 陆恒逼迫厂长认罪
- 使用规则豁免权打破规则
- 回到现实，能力进化
- 核心悬念：诡异是否会蔓延到现实？

### 核心悬念系统

| 悬念 | 埋设位置 | 揭晓位置 | 功能 |
|------|---------|---------|------|
| 弹幕是否可信 | 第1章 | 第10章 | 驱动主线 |

| 诡异是否入侵现实 | 第11章 | 第12章 | 系列化钩子 |

### 规则系统（世界观规则）

1. 副本规则：黑板上规则有真有假，违反真规则会死
2. 弹幕规则：来自另一个维度的观众，信息有真有假
3. 通关规则：不是消灭 Boss，而是让 Boss 承认罪行

# 第三本：《婆婆逼我养外孙，我直播掀桌离婚》

- 婆婆带外孙上门逼宫
- 林晓棠当场开直播
- 直播读账本、播放录音，舆论反转

# Novel-OS 项目使用说明

## 文件放置建议

如果要用 Novel-OS 写其中一本书，建议创建如下目录结构：

```
books/毒杀重生后我用医书屠了贵妃/
├── analysis.md       # 本文档中的故事结构分析
├── characters.md     # 本文档中的人物双轨表
├── rules.md          # 本文档中的写作硬规则 + 世界观规则
├── outline.md        # 本文档中的详细章节大纲
└── chapters/         # 生成的章节正文
```

## 与 Novel-OS 流水线对接

1. `Director` Agent 读取 `outline.md` 生成本章任务卡
2. `Writer` Agent 按任务卡写初稿，遵循 `rules.md`
3. `DeAI Interceptor` 检查他字密度和禁用词
4. `Auditor` 审计字数、人设一致性、伏笔回收

## 关键质量门禁

- 字数：1500-2200 字/章（短篇可按此标准）
- 他字密度：≤ 8%
- 禁用词：然而/不得不说/众所周知/突然/竟然/原来/与此同时
- 每章必须有：冲突单元、爽点/钩子、情绪推进
