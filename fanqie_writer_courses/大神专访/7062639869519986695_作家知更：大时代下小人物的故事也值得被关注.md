# 作家知更：大时代下小人物的故事也值得被关注

2022年2月，北京冬季奥林匹克运动会火热进行中。

番茄小说网上，也有一些以冬奥为背景的网络文学作品。这一期，我们就邀请到《幸福在家理》的作者知更，让我们跟着作者们一起走近冬奥，体会冰雪运动题材作品的魅力与激情吧！

知更，本名刘家希。北京市作家协会会员。作家，编剧，自由撰稿人。

长篇小说《双生》，获2019年度中国作家协会网络文学重点作品扶持奖；

长篇小说《幸福在家理》，获2021年度中国作家协会网络文学重点作品扶持奖；

中篇小说《你睡了，我醒了》入围 2018 年第二十一届上海电影节百部官方推荐作品。

爆姐：作者大大简单做一下自我介绍吧。

作者知更：我是《幸福在家理》的作者知更。90后，专职码字，偶尔编剧。开始写作是在五年前，我当时在英国伦敦帝国理工大学留学，随手开了一个记录留学生活的散文专栏，没想到反响还不错。就这样，我有了自己的第一部作品《英漂第五年》。

就是从那时起，我开始走上写作这条路。到今天，已经有了三部长篇小说，一部短篇小说，一部散文集。其中小说《幸福在家理》和《双生》还获得了中国作家协会网络文学重点作品扶持奖，并出版改编，即将搬上荧屏。

其实，我从来没有想过自己有一天会成为作家。

爆姐：哈？那你是怎么走上写作之路的？

作者知更：高中时，我的梦想是做医生，之后的好多年，我也是一门心思往“当医生”这条路上走。我是一个特别执着的人，不撞南墙不回头的那种。就因为高中时特别想学医，所以高考填志愿时孤注一掷，只报了北京大学医学部一所，完全没给自己留后路。不过还好有惊无险没有落榜，只不过差了几分没进理想的临床专业，而是被调剂到了药学。我当时以为，药学就药学吧，应该也差不多，也算是离梦想又近了一步。但后来才发现，失之毫厘谬以千里，从那时候开始，我就离当初的梦想越来越远了……

因为没当成医生，之后我也经历了一段迷茫期，没有勇气从头开始，也没有办法改变现状。在这段时期，我尝试了很多新的东西，比如画画、写作，去发现自己的兴趣和特长，也很幸运地发掘了自己写作方面的潜力。我不认为之前的经历都是没有用的，正是这样的人生选择和心路历程让我对生活有了更多的感悟，也在创作中萌发出更多灵感。

爆姐：创作源于生活。介绍一下在番茄创作的这部作品《幸福在家理》吧。

作者知更：《幸福在家理》这本书是我所擅长的现实题材。故事发生在2022年冬奥会的背景下，讲述了北漂女孩宋家理辞掉工作回老家的另类奋斗史。在事业瓶颈、爱情流产的双重打击下，她选择逃离“北上广”，做快进生活中的逆行者。从无奈躺平到咸鱼翻身，最终创业追梦重拾幸福。

故事讲述的是冬奥背景下的小家幸福，是一个聚焦小人物的故事。而小人物的成长必须在大时代的背景下才能成立。

![image](http://p3-novel.byteimg.com/origin/novel-static/0a4ed2825ff26a908863965562e0d304)爆姐：为什么会想到创作以冬奥为背景的作品呢？

作者知更：我出生在张家口，长在北京。所以我对于这次冬奥会有种特别的家乡情怀。时隔多年重游张家口，看到了城市的巨变，和因冬奥带来的发展。我觉得在这样一个历史节点，应该有更多目光从大都市转移到小城镇身上。

我们以往看到的作品中，大城市和都市白领经常是故事的标配，但我觉得，小镇青年和小城也值得被关注。所以我就想到结合冬奥会这个高光时刻，把我们身边小人物的故事记录下来。于是就有了《幸福在家理》这部作品。

故事中，冬奥会象征着梦想，主人公宋家理和她的家乡喜镇象征着生活中平凡的你我他。谁能想到就是这么一个不起眼的小城，竟然有一天会在全世界的瞩目中迎来梦想中的冬奥运。而投射到小说中的人物身上，处在人生低谷的宋家理在落魄回乡无奈躺平时，又怎能想到有一天她会自己当老板，成为引领乡邻建设家乡的带头人。

我希望通过这个冬奥背景下的故事，向读者传递一种信心和力量，让那些和主人公宋家理同样处于人生迷茫期的朋友们看到未来和希望，将家之小梦与国之大梦紧密相连。

爆姐：这个想法太棒了！分享一段作品中你最喜欢的内容吧。

作者知更：我个人最喜欢这一段。这一段讲的是一对曾经恋爱8年的情侣在分手3年后的重逢。当一事无成的宋家理遭遇事业有成的前男友，她的心情会怎样呢？不服输，却输了；生怕自己看起来过得不好，但又确实过得不好。所以她撒谎她吹牛，说自己当了老板，又开餐厅又开客栈——餐厅是别人的，客栈还是没影的事。但不管怎么说，她不能在前男友面前丢人。

![image](http://p3-novel.byteimg.com/origin/novel-static/962a0b081518afe1f068e178521c18bc)![image](http://p6-novel.byteimg.com/origin/novel-static/bc812f65a0be255552e20141e450efbc)我很喜欢宋家理的真性情，敢作敢当，敢想敢干。

她的日子似乎有今儿没明儿，但到了明天又能想到新的办法对付过去——但她其实不是对付，是不凑合。

在小说前半段，宋家理是个丢在人群里都不起眼的小北漂，也许我们每天在地铁里马路上与千千万万个这样的女孩擦肩而过。但她却比人群中的大多数都坚强、大胆。作为都市白领，一位坚守底线的设计师，她敢于抡起大锤把不合心意的一面墙砸出洞来；作为小镇青年，她陷入泥泞的低谷，却敢于坦然脱掉高跟鞋换上趿拉板。人生拐入死角时，她敢于停下脚步做一条咸鱼；舒服的躺平后，她也敢一跃而起成为跃龙门的锦鲤。

人生不是一马平川，也没有注定的成败。渺小的我们究竟是咸鱼还是锦鲤，需要自己决定。一个在人生路上敢于驰骋、敢于刹车、也敢于转弯的人才是真正把控自己人生的人，是所有人羡慕的对象——而故事中的宋家理就是这样的人。

爆姐：好喜欢宋家理的人设，这类现实作品的人物形象刻画往往要求很高，你在创作时一定做了不少功课吧！可以分享一些此类作品的人物设置技巧吗？

作者知更：没错，现实类作品重在真实，因此对人物设置有更高的要求。我在构思时，总会在人物上花大把时间。我不喜欢写十全十美的人，因为那不真实。所以玛丽苏、白莲花和圣母绝对不会出现在我的故事中。

我希望写出的每个人物都是在现实生活中立得住脚的人，站在阳光下有影子的人。他们身上也许有各种各样的小毛病，但又不乏别样的闪光点，既要有普遍的社会代表性，又要适当拥有戏剧化的个性。故事中的每个人物都应该有自己的思维和视角，而不是被写作者操控的傀儡。有了这一群活生生的人物，便会自然而然碰撞出有趣的故事情节。

写作中我有机会通过不同角色的视角看问题，我很享受在写作时与人物交流的过程，甚至可以从笔下的人物身上得到些不曾有过的感悟。

现实题材的本质是从生活的平凡中寻找闪光点。笔下的故事仿佛发生在我身边，却又比我的生活更加精彩。

爆姐：你跟主人公宋家理是否有相似的地方？

作者知更：我在创作的时候其实并没有把自己作为人物的原型，甚至从来没有考虑过这个问题。但现在回看《幸福在家理》的完本，宋家理身上确实有一些我的影子，而这个北漂女孩的人生抉择，可能也是你我正在经历或者曾经经历过的。

就好比我在英国留学期间，也面临和“北漂”相似的抉择。是该留在异乡打拼，还是回到家乡寻梦。究竟哪条路才是对的选择？我想这个问题可能没有一个准确的答案。

而我最终还是选择了回国，和书中的宋家理一样。正是这段经历在创作上给了我启发。我并不觉得宋家理 “逃离北上广”的选择是一种落荒而逃的失败。每个人都有选择人生的权利，不管是咬牙坚持还是转身离去，遵循自己内心意愿作出的决定才不会后悔。

我们没有办法同时体验两种平行人生。我们能做的只有为自己的选择负责，为自己的人生做主。并且条条大路通罗马，走投无路时再迈一步就会柳暗花明，咸鱼翻身就可能变成锦鲤，小城也能走向世界，小人物也可以有春天。希望这部正能量的作品可以给读者带来力量。

故事的主人公虽然不是我自己，但作为主人公的同龄人，我了解她的苦恼，想她所想，及她所及。她像是我镜子里的分身，又像是我现实生活中的朋友，更像是穿越到书中的读者。我力求让笔下的人物更好的贴近生活，直面现实中的心灵困境，在切中社会热点和人群痛点的前提下，和读者一起探寻出一条披荆斩棘打怪通关的路。

爆姐：创作这部作品的过程是不是也挺“披荆斩棘”的？有什么经验和心得可以分享吗？

作者知更：没错，我在这个故事构思上花费的时间之长，让编辑一度以为故事要流产。但回头看，我每一个故事大纲构思都要经过几个月，总是改了又改，非要磨过好几次才能满意。有时甚至和写完正文的时间差不多长。

但我并不觉得这是在浪费时间。一部动辄几十万字的小说，一定要在动笔之前打好基础。而大纲和人物就像盖房子的地基，只有地基打得稳，之后的一砖一瓦才能砌的牢固。

我们习惯把写作叫做“码字”，但它又不是单纯的文字堆叠。想好再落笔，比落笔后再改要容易得多。相信每一个作者都会遇到卡文或者缺少灵感的时候。我在这种时候不会逼自己非动笔不可，而是会停下来分散一下注意力。不管是看书、上网还是出去走走，你不知道哪个瞬间就会成为灵感迸发的开关。

作为一名写作者，有个好处是，不存在所谓无意义的事，人生中的每一刻、每个经历、每一次喜怒哀乐都可能成为创作的灵感。现实生活的平凡故事中也存在很多出人意料的惊喜和令人难忘的乐趣。有时候甚至会比编出来的故事更戏剧化。我喜欢观察生活，喜欢听别人讲故事，也喜欢滑手机上网。在最平凡的生活中可以收集到最有趣的事，再经过笔头的加工润色，就变成了有趣的故事。

爆姐：还好这个故事没流产！编辑一定也很开心吧！

作者知更：嗯嗯。不止是开心，编辑还对这部作品寄予厚望。创作《幸福在家理》这部作品的时候正值建党100周年，中国作协和文联有很多针对现实题材作品的扶持和奖项。编辑对我的作品信心十足，鼓励我申报了几个重要奖项。也正因如此，《幸福在家理》有幸获得了2021年中国作家协会网络文学重点作品扶持奖，以及第五届中国网络文学大会+优秀影视IP作品。

此外，在平台支持下，《幸福在家理》已经与陕西太白文艺出版社协议出版，纸质书预计于年底问世。

知更冬奥题材作品《幸福在家理》可在番茄小说APP免费阅读。
