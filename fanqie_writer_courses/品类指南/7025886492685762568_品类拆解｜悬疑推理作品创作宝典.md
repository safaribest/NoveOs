# 悬疑定义：

悬疑小说的核心在“悬疑”两个字。“悬”是悬念，“疑”是怀疑，“悬疑”指的是因未知而产生的怀疑心理，因此，悬疑小说是以解决悬念、探索未知作为主要剧情的小说类型。

悬疑小说往往具备三大特点：曲折的情节、强烈的悬念和严谨的逻辑，但其内容和题材都相对自由广泛，不一定要有让人感到十分害怕的场景和气氛。

# 悬疑分类：

## 灵异风水：

灵异风水小说是流行于现代社会网络文学中的一种新兴小说文体，一般以神鬼事物为题材，往往出现鬼魂、异变生物等惊悚元素，展现猎奇的故事情节，给予读者的刺激感较强。但与单纯以惊吓作为卖点的鬼故事不同，灵异风水小说从诞生起就包含着朴素的因果思想，旨在通过一系列因人的贪念或其他欲望而引发的灵异事件，揭示”善有善报，恶有恶报”的观念，引导读者树立正确的价值观，积极行善，具有一定的教育意义。部分以盗墓、风水、寻宝、探险等为主线的作品，还往往将灵异事件与考古知识、周易八卦、中国民间传说以及地方风俗等相结合，为展现中国传统文化提供有趣的切入点。代表作品：《我有一座恐怖屋》、《魔临》、《鬼吹灯》、《盗墓笔记》等。

## 推理探案：

推理小说是以推理方式（以严密科学的逻辑思维和推理手段）解开谜题的一种小说。一般来说，推理故事都含有凶杀案与侦探，但是此处“侦探”也不一定是警察、探长等本人，案件也不一定完全以凶杀为主要剧情走向，也存在是寻找失物、遗落的宝藏、还有离奇事件等为主。

探案小说主要是以刑事案件为主线，内容上会主要以刑事案件的侦破为主要情节，大部分主角会是警察、侦探等本身职业是跟探案相关的。代表作品：《心理罪》、《法医秦明》、《无证之罪》、《长夜难明》等。

# 写作技巧：

## 灵异风水：

背景设置：塑造恐怖的氛围、渲染的手法和情绪要到位，读者才能被吸引。总的来讲，有三种手法比较常见：

人称转换：以第一和第三人称为主。第三人称一般用于对故事背景、事件发展程度、前因后果来龙去脉的交待，当需要对恐怖气氛和人物内心进行细致刻画、以增强读者带入感时，往往采用第一人称。信息不对称：恐怖氛围的营造不仅来自于作者的描述，有时也来自于读者的自由想象。因此好的作品在描写恐怖氛围时，需要通过刻意制造作品中人物与读者间的信息差，来让紧张感升级。比如《我有一座恐怖屋》中的某次事件，陈歌被剥夺了视力，独自一人待在鬼魂出没的居民楼里，从读者的角度能够得知鬼魂离陈歌还有多远、他面临着哪些危险，但陈歌对此却一无所知，读者时刻替他捏着一把汗，紧张感也就油然而生。反差对比：除了对灵异环境本身的描写外，有时也可以反其道而行之，以看似轻松惬意的表象来反衬出背后潜藏的恐怖有多骇人。最能够刺激到读者的往往是生活中随处可见的平常之处突然变得不平常，这也是恐怖片热衷于以卫生间、医院、学校等作为背景的原因。比如《地狱电影院》的副本《异度教室》，一方面主角们在真实的恐怖剧本中挣扎求生，但另一方面他们又被迫体会真实的高中生活，哆啦A梦漫画、金庸、古龙、老式的放映厅、上学路上的三五成群、放学后的补习······这些本该是每一个8090后高中生都习以为常的生活经历，但恐怖偏偏就降临在这些时刻。这种反差感足以让每一个读者在猝不及防间汗毛倒竖。人物设置：灵异风水的主角在事件发生前，会比较普通（或者是自己并不知晓），因为一件事而改变了原本的人生轨迹，例如发现了自己身世的秘密、意外得到了金手指、莫名卷入神秘的诅咒等，进入事件后打开了新世界的大门。但主角的生活背景可以普通，自身的性格必须随着恐怖程度的提升或事件的不断深入而有显著成长。比如《我有一座恐怖屋》的主角陈歌一开始只是一个鬼屋老板，在开篇并没有表现出过人的胆识，需要靠鬼魂来保护自己。但随着黑色手机发布的任务难度不断升级，他游走在人鬼之间，一次次识破冥胎设下的陷阱，从被保护者蜕变成了保护者。事件设置：灵异小说因为需要悬念，通常会以离奇的小事切入，环环相扣，事件不断升级。例如《凶宅笔记》，就是主角“我”因为从倒卖凶宅中发现了商机，跟好友一起发现一件件事情后被套入了内网。虽然恐怖灵异小说允许出现超自然力量，但小说本身的基本逻辑依然要遵循，不能简单把事件的起因、发展和解决都推给超自然力量。故事的真相应当与作者对故事的讲述同样精彩，不能出现离奇事件频发，最后发现一切都是主角的一场梦，梦醒了什么都没有了这样的情况。## 推理探案：

背景设置：以凶杀案、诡异事件为主。开篇故事就要离奇，才能抓住人，每层都环环相扣，后续不断反转。案件可以是单一的故事，类似单元剧那样，比如《福尔摩斯探案集》、《法医秦明》都是主角接触的各种案件为主，其主线主要在人物关系上，并非事件本身的升级。人物设置：不管男女频中，探案类的主角一般都需要有较高的智商、敏锐的观察力，抓住案件中的疑点。但是这类故事对主角人设要求比较高，不能以千篇一律的冷静机智来概括，应当突出与其他推理小说主角不一样的地方。比如高智商低情商的福尔摩斯，或者是《他来了请闭眼》中清冷孤僻的薄教授。凶杀案真凶或诡异事件的幕后黑手应当有一定的迷惑性，不能设置得太明显，一出场就明白谁是真凶会使读者丧失阅读兴趣。必要时可制造烟雾弹，让读者在确定与不确定之间反复摇摆，从而吸引其继续阅读。事件设置：推理探案类讲究事情的关联性和逻辑性，故事中不停发现新的线索，但前面的道具也并不能丢掉。这类的案件最终解决一般都会出人意料，又要在情理之中，如果最后变成偶发事件/意外事件，难免会有些虎头蛇尾，就会不符合读者的期待。

# 注意事项：

不要宣传封建迷信，如剧情需要出现封建迷信元素，不做细致刻画，应一笔带过并将其作为批判对象或反面例子。尽量不要出现陋习，部分陋习如重男轻女、吃绝户等可作为批判对象存在，部分陋习如食用同类的尸体、陪葬人殉等不能出现。女性向悬疑小说特别注意，不能出现冥婚、结阴亲、怀鬼胎等元素；男性向悬疑小说特别注意，不许出现冰恋元素。血腥场面描写应尽量模糊化处理，不要太过细致，不许出现屠城、毁灭世界等反人类场面。灵异风水类小说允许出现神鬼事物，但不能出现真实地名、政府机关及领导人名称，不得借作品影射现实政治，也不能出现如政府领导人被超自然力量杀害等场景。不允许以悬疑之名行色情之实，涉及到浴室、卫生间、卧室等地点的描写，如非剧情必要，不应特别指出女性角色是否裸露并进行细致刻画，也不能以鬼魂的视角来写。
