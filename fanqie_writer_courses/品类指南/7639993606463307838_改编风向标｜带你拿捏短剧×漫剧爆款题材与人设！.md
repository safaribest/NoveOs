# 一、短剧改编方向

# （一）女频

近期爆款女频短剧改编以打脸逆袭和女性成长为主，题材主要集中在破镜重圆，婚后日常等类型上，内容风格偏好甜虐交织，双向救赎，治愈甜宠等方向。

例1：

短剧《野荷荒长》（原著《野荷刺》）以“校园初恋-久别重逢-破镜重圆”为主线框架，讲述男女主因阶层差异和性格差异反复拉扯的爱情故事。

感情线出彩的同时，女主人设鲜明有看点：虽身陷原生家庭的泥沼，性格略显自卑，但不放弃每一次自救机会，靠自己的努力，不断成长，人物形象独立出众，极具人格魅力。

作品简介：

【女性成长文，男生前期有青梅竹马的准女友，久别重逢】

隋荷是一株杂草，她贫穷，虚荣，敏感，有心机，她唯一的愿望就是考出去，自己赚钱自己花。

她知道，沈确和她不是一个国度的人，他家境殷实，朋友多，学习好，受女生喜欢……

他永远不明白她到底是为了什么在拧巴，他永远不懂。

所以她不要他的喜欢，她不要他高高在上的喜欢。

例2：

短剧《晚星轻摇落沉舟》（原著《新婚当天休夫！踏碎凤冠上马回府》）讲述：现代女主穿书古代，为逆天改命，当街退婚，并救下街头痴傻男主招为赘婿，殊不知，男主是中毒失忆太子，后男主恢复神智，默默守护女主。

故事以婚后日常为主线结合穿书逆袭的故事背景，添加双向救赎、治愈甜宠等高热情感元素，兼顾剧情线上的虐渣逆袭和感情线上的甜蜜互动。

作品简介：

【穿书＋救赎＋空间＋双洁】【哭唧唧英俊傻夫君 X 霸气女主温馨沙雕日常】

穿书成新婚被渣男气死的女配，她脱嫁衣踹渣男招傻子为赘婿，人人笑她眼瞎，谁料婚后傻子摇身一变竟成了尊贵的大皇子，所有人都傻眼了！

夜里，她揪着傻夫君耳朵冷笑："乞丐变王爷无妨，可你若是从傻子变成了正常人，那你死定了！"

已恢复正常的傻夫君心想，要他这朵高岭之花每天装憨装傻哭唧唧要亲亲要抱抱，他真的好为难啊！看来，是时候骗她生个萌宝了……

# 内容建议：

· 题材类型：注重题材微创新，在破镜重圆和婚后日常等经典题材的基础上，寻求设定和剧情上的差异化，例如：久别重逢+婚后日常。

· 元素设定：可尝试在感情线中，融入救赎，治愈等热门元素，在故事主线中集合亲情线元素，让情绪更接地气，更能引发读者共鸣。

· 人物设定：男女主cp设定中，强反差组合依旧是核心看点，可重点从人设反差延伸，丰富感情拉扯的层次感，其次，在主角身份和主角性格的反差上也可以做重点设计，增强内容吸引力。

# （二）男频

近期爆款男频改编以逆袭崛起和打脸虐渣为主，题材集中在历史权谋上，其中在历史权谋中结合领地经营元素，着重刻画人物反差的作品表现尤其出色。

例如：

短剧《大夏荒州王》（原著《我，腹黑皇子，天下美人都想弄我》）现代兵王魂穿炮灰皇子，凭借强悍实力和穿越者的超前思维，粉碎敌人阴谋，平定边疆战乱，绝地逆袭成威震四方的大夏荒州王。

全剧融朝堂权谋、疆域征伐，男主从困守荒州到称王一方，从逆袭变强走向建功立业，故事剧情极具层次感。

作品简介：

他是帝国的废物皇子！

但，皇帝老子要杀他，太子要杀他，皇子们也要杀他，就连天下第一美女也要杀他！

为活命，他练无敌大军，建世外桃源，威压诸子百家，引得四方能臣武将和隐世高手纷纷来帮，让天下美人都想做他的新娘。

谜团重重的夺嫡路上，他也揭开了一段段诡异的历史真相……

# 内容建议：

· 题材类型：重点关注都市异能，历史权谋和年代题材，在常规题材设定下，重点关注剧情的丰富度。

例如：

在比较经典的历史权谋智斗题材中，不止可以写朝堂上波谲云诡的文斗，还可以结合平定边疆豪气冲云天的武斗，更可添加在地方上的经营种田元素；

情绪张弛有度，剧情爽点表现方式多样，也能让故事架构更丰满，故事地图更完整，代入感更强。

· 元素设定：多元素融合趋势明显。

例如：

废柴流可结合权谋，微悬疑，系统，逆袭金手指等等，在把握核心故事主线的基础上，合理的进行多元素融合，可以更好的丰富故事看点。

· 人物设定：大佬装废柴，迪化，反差等都可尝试，重点是人设要与剧情设定结合，两者互相推动，人物细节要饱满。在日常互动的剧情推进中，富有趣味性的人物设定对读者更有吸引力，也更利于后续改编。

# 二、漫剧改编风向

# （一）女频

女频漫剧改编的主线设定仍旧以打脸逆袭为主，漫剧题材主要集中在家庭伦理和养崽种田等内容上，结合女性成长、打脸、甜宠、温馨治愈等元素。

例1：

《五十岁，我的人生刚刚开火》中，主角是早已年过五十的母亲身份，遭遇女儿和女婿的伤害背叛后决定重新来过，凭借昔日国宴大厨的手艺谱写新的人生。

前后不同人生选择的重启，强调了女性的独立与成长，诠释出“母亲”只是一个称谓，前后剧情层层递进、爽点连贯，主角人设前后反差感强。

例2：

《白鲤劫，破茧重生》中，女主是重生一世的白锦鲤，重生后本想好好活下去，却依然饱受扭曲的“血统”观点毒害，最后通过自己的不断努力成功觉醒祖龙，打破血统的畸形阶层划分。

例3：

漫剧改编原著《一纸休书，疯批揣娃带娘家暴富》中，女主原身是只看重皮肉的败家女，主角在魂穿后选择离婚带领全家暴富赚钱、摆脱贫穷。

作品简介：

【空间+基建+女强+爽文+权谋+乱世+天灾+虐渣+女帝】

刘月月地府走一趟，醒来发现穿越。

渣男一家极品想让她净身出户，她成疯批麻溜拿上休书，要回一半嫁妆，带着儿女赶紧走人。

奈何，原主臭名远扬，窝里横，外头欺。

娘家嫂子不待见，寡妇上门找事，舔狗想要占便宜。

看来想要立足必须搞钱，疯批人设立起来，吵吵不行就动手，怎么顺心怎么来！

带上在王殿顺回来的聚宝盆，加上她上一世的生意经，她不暴富谁暴富！

只是没想到，不就是休夫虐个渣，怎就混成了女帝？

# 内容建议：

· 题材类型：现言脑洞、宫斗宅斗等题材仍旧占据一定市场，同时可以关注养崽种田、女性成长/互助、养崽种田等热门题材故事设定。

· 元素设定：养崽、种田、逆袭打脸、女性成长成为热点话题，可参考。

· 人物设定：仍旧强调主角崛起，可侧重魂穿脑洞、重生打脸爽、复仇虐渣以及适当甜宠方向，建议重点去做高信息差反差感打脸。

例如：

《末世战神穿农家，靠万能空间养崽暴富》中，女主魂穿后从末世战神转变成培养孩子长大的母亲，最后凭借自己努力与孩子之间消除芥蒂、走上富贵路。

# （二）男频

男频漫剧改编思路以男频爽文套路为主，题材主要集中在乡村种田、脑洞、战神归来、赘婿等内容上，强调打脸虐渣思维、小人物成长逆袭打脸爽、全程高能爽、白眼狼+恶有恶报等情绪的代入。

例1：

《1987，砖瓦厂食堂散伙》中主角本为照顾贫困的好友一家，尽心帮对方打开了承包食堂的销路，最后却被好友踢出局，此后，主角以旁观者的身份，看着昔日好友因贪婪而重归贫穷，全剧展现因果报应，白眼狼最终恶有恶报。

例2：

漫剧原著《桃源乡村小神医》中男主英雄救美却遭人陷害后归家，面对恶霸刁难、父母辛酸，男主凭借自己在外多年所学医术一步步带领父母脱贫致富，甚至最终打脸恶霸，成功带领全村发家致富。

该剧融合了神医人设+乡村种田的双重设定，男主归家后展现出来的家庭担当以及个人胆识极具个人魅力，人设鲜明，打脸恶霸、发家致富、神医打脸等剧情设计爽点充分、看点足。

作品简介：

小子白云飞英雄救美，却遭人所害，大难不死必有后福，习得逆天功法，掌神奇医术，从此逆天崛起，恩要还，仇要报！

# 内容建议：

· 题材类型：都市、年代题材的占比逐渐增加，出现题材回春的情况，同时玄幻、脑洞、高武依旧效果不错。

· 人物设定：高位打脸、小人物逆袭可以大胆尝试，全程高能人设近期较少，但厚积薄发型主角设定日益增多：

无论是在乡村种田发展遇阻时挺身而出、捍卫“邪不压正”的乡野能人；

还是隐姓埋名后又重新崭露头角的归来战神；

高反差、高期待、高爽感的设置增强男主人格魅力。

· 元素设定：战神归来、赘婿有回春迹象，乡村种田+发家致富发力明显。

例1：

《那日，我买下了全村最破的院子》中主角被未婚妻一家当众凌辱后，买下旧地主家的破败宅院，随后偶遇机缘，凭借机智逆境谋生，不断壮大。

纵然屡遭恶势力打压，亦能能坚守本心，谋求生机。主角一步步崛起、虐渣打脸，爽感足。

例2：

漫剧原著《天道酬勤：开局联姻落魄千金》中主角觉醒后成弃子，联姻落魄千金，却因天道酬勤终获圆满，惊艳众人。

作品简介：

【杀伐果断＋面板+升级流＋无视瓶颈】

沈长青一朝觉醒前世宿慧，喜提天道酬勤天赋：

可无视境关，只需经验足够，便可直接突破，一证永证！

可还没来得及大展拳脚，就被亲生父母当成了替天才弟弟联姻的弃子，对方是家道中落，负债累累的“落魄干金”。

本以为是个天大的火坑，但当神清骨秀，天资不凡的绝美妻子出现在眼前，再感受着体内无视境关、水到渠成的修为时….他笑了。

“联姻？可以！”

“但老登，得加钱！”

从此，灰雾笼罩的修行绝地，多了一个只靠肝就能一路升级的赘婿：

别人破境卡关欲生欲死？沈长青埋头苦练，境关？那是什么？经验满了自然破！

顶级剑法无人能成？他天道酬勤，肝到圆满惊呆观主！

貌美明理的妻子恍然发现：“原来我夫君竟是绝世天才！”

沈家的天才弟弟等着看笑话：“哥，李家的火坑暖和不？”

沈长青默默拔剑：“这一剑，二十年的功力，你挡得住吗？”
