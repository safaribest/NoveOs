# 科幻定义：

用幻想艺术的形式，表现科学技术远景或者社会发展对人类的影响。科幻作品通过科技、未来等方面的幻想，创造出让读者感到陌生和新奇的世界观和社会背景设置，创造出新奇感和陌生感。科幻的关键是新奇感。

# 科幻分类：

硬科幻：以严格技术推演和发展道路预测，以描写极其可能实现的新技术新发明给人类社会带来影响的科幻作品称为硬科幻。《基地》《太空漫游2001》《三体》等。

软科幻：科学技术和物理定律重要性偏低的科幻，更注重社会发展，如《沙丘》《星球大战》等。

科幻网文细分类型与写作技巧：

网文科幻偏向软科幻的作品更多，科技严谨性要求不高。把“科技”或者“未来”作为想象的背景，用来展现爽点，偏流量向，爽点和网文基本相通。各细分类型如末世文，无限流，星际文等等，基本以软科幻为主。

## （1）末世文怎么写：

末世文是网文科幻传统类型。通过描写现存社会秩序的崩溃，以及社会秩序崩溃之后的新的末世社会的发展，创造出陌生感和新奇感。人性在社会崩溃之后的变形，搭配人在末世中进化变异的特殊能力，能创造出与现实完全不同的社会和人物关系。如一本老的末世文（随风飘摇的《蹉跎》）中人物互相之间的祝福语是“祝蘑菇丰收，老鼠满仓。”因为蘑菇和老鼠是末世的食物来源，写出的就是末世中让读者有陌生感的人际关系。

写作关键：末世文要写出创新，用创新点抓住读者。末世文要让读者感到陌生感与新奇感，末世文整体数量大、同质化作品多，要创造陌生感和新奇感就需要创新，与其它末世文拉开区分度。

创新点：

类型创新。举例：灾变文，淡化丧尸元素。设置多样的灾变类型，每个灾变类型相当于一个大副本。不断发生新的灾变，每有一个新的灾变发生就相当于进入一个新的副本。站内举例《灾变：除我以外，全球人类失明了》《全球灾变：开局建造诺亚方舟》。

金手指创新：变异进化获得能力的方式，结合流行元素，如签到等。

副本创新：前文提到的灾变副本，以及无限流副本、求生文副本等，《诸天末世在线》。

## （2）黑科技文怎么写：

黑科技文一般是都市背景，有一定都市爽点，金手指是未来黑科技。

写作关键：抓住黑科技在都市产生的爽点，同时写出黑科技对社会的影响和推动，而不只是用黑科技在都市装逼。黑科技推动现代社会走入未来社会，才能写出科幻的陌生感和新奇感，与众多都市文区分开。

黑科技金手指可以与其它类型结合，例如黑科技+末世文灾变文。

## （3）星际文怎么写：

星际文一般属于软科幻，不强调科学技术的原理，而是把星际作为大背景去想象星际文明的不同形态。除了网文之外，如《星球大战》等著名ip也都属于星际软科幻，这种题材又被称为“太空歌剧”。

写作关键：星际文中，每个星球相当于一个世界，情节的核心是写出各种世界之间的互动。每个世界实际上能起到不同副本的作用，给每个副本设置不同的特色、种族、势力、文明形态，为读者创造新奇感，吸引住读者。

将不同星球副本串联起来的星际文明，往往有从低等到高等的发展过程，类似修炼升级的体系。星际文明可以设置不同形态，如虫族、智能机械等不同的文明形态都很适合发挥想象力。

太空歌剧类的星际软科幻可以加入高魔元素，如星球大战系列的原力，星际争霸的幽能，战锤系列的灵能，等等。星际文在有未来感的同时，往往有奇幻感，重视远古遗迹、上古神明之类更偏奇幻玄幻的设定。

站内《深渊星芒》即为类似作品。

# 分类总结

科幻题材的核心是用创意抓住读者，为读者制造新奇感，吸引读者阅读。末世文，黑科技文，星际文等子分类，各自的重点也都是用有新奇感的设定抓住读者。科幻的核心是想象力，需要想象出有新奇感的世界，并能吸引读者沉浸其中。
