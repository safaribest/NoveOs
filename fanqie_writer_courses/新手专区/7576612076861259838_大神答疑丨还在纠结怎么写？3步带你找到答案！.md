# 一、写作最重要的是什么？

当然是写啊！！

别人都在观望，你已经动笔，你始终领先你朋友一步。写了不一定赚钱，但不写一定不赚。

好了，讲过最重要的，我还要讲一下次要的。

### 1.1 好故事胜过好文笔

拿到这个问题的时候，我其实也愣了一下，这个问题以前还从来没想过，就是闷头写。但是今天我仔细想了一下这个问题，觉得可能就写小说而言，最重要的应该是故事性。

小说本质上就是讲故事，故事性要强，事件要跌宕起伏。

我见过很多新人在刚入行的时候陷入误区，会强调文笔，用大量华丽的辞藻来堆描写。很多新人的描写说实话我是非常钦佩的，有的形容词我甚至连字都认不全……看到了就默默打开我的浏览器，查一查学习学习。

当然，并不是说文笔不重要，好的文笔无疑是锦上添花的存在，有时候我碰到厉害的成语和描写也会去学一学。

但小说的故事性才是一篇小说的基础，把一本小说比作盖房子，故事性就是房子的框架，文笔就是外观。如果说房子只是一个茅草屋，你非要给它贴瓷片，房子能承受住吗？

读者看第一章，看了半天还没进入正题，会很快弃文的。

### 1.2 新人突破口：构思新鲜的故事

从写小说以来，有太多新人问我，怎么提升文笔，怎么提升代入感等等。

大家会去思考这些问题，归根到底其实就是想更多的人能看到自己的书，从而提升知名度和收入。

新入行的朋友不妨将这些问题先暂且搁置一下，先想好怎么讲故事。

大家小时候有没有听家里的长辈讲故事？小时候我外公经常给我们讲故事。

比如有个小故事，讲的是小朋友上山打柴，给人参娃娃衣服上别针的故事。

对那时候的我来说，天呐，这也太新奇了！人参竟然能变成娃娃？我听得津津有味。

要说我外公多有文采？其实也没有，就是很接地气的讲法，毕竟讲得太复杂的话，当时的我可能也听不懂。

写小说其实也一样，每天发书的人会有多少？没有一千也有八百了，要怎么从这么多书中杀出去？其实就是看谁的书故事性更强，更新鲜，也就是大家说的噱头。

大家看那些热门的小说，看过之后再回忆起来都有非常令人难忘的故事情节。

新入行的朋友们，与其用大量时间研究文笔，不如想想有没有什么新鲜的故事点。

新人时期思维还没有被圈子里的条条框框限制，将自己天马行空的想法写出来或许更容易出成绩。

### 1.3 行动与坚持是成功的起点

这个圈子有一定的门槛，但想迈进来也简单。还是那句话，写小说最重要的事情就是先写，先走进来，文笔节奏还有什么情节过渡等等，写得多看得多也就明白了。

# 二、如何选择合适的题材？

这个问题其实应该很多人都遇到过，看到某一题材很火爆，自己想去写，结果发现根本写不出来。很多人在这一过程还会疯狂内耗，焦虑，很大原因就是因为题材和自己的性格不符。

### 2.1 选对题材：兴趣和性格是关键

有一句话叫做，淡人写不了情绪文。比如我们本身的性格是一个怕跟人起冲突，在外边不太敢表达自己，这样的性格来写情绪文，根本驾驭不了。或许转变一下思路，去写日常文，种田文这种类型会更适合。

选择合适的题材还要看自己的兴趣点在哪里，比如后悔流，后悔流追妻火葬场在网文圈子里也是一个很大的类别，养活了无数作者。

但有的人就不太能理解，为什么要让他后悔？人都死了，后悔有什么用？！这种文究竟爽在哪里啊？！

会有这种想法，就说明这种类型的文不适合自己，也没必要去一味的迎合。掌握不到这种类型的爽点，写出来的东西节奏和这类型文的读者不同频，就很容易扑街。

### 2.2 从读到写：你的阅读史就是最佳指南

建议大家还是以自己的兴趣为中心，再向外拓展。

简单讲，就是你喜欢看什么类型，就写什么类型。

时间花在哪里，收获就在哪里。

喜欢看的类型，大量阅读就是创作的根基，你能更好的把握爽点和读者的情绪点，写起来也更加顺手（比如日常对话、动作场面、心理描写）。

这样也能保证基本产出效率，毕竟大家都知道的嘛，写小说每天要更新四千字才有全勤，就算不奔着全勤去，至少也得每天更新，三天两头断更，读者也都跑掉了。

再结合我第一点讲的，写自己喜欢看的类型，加入故事性，也就是一定的噱头，会更好的发挥出自己的创作优势。

### 2.3 纠结题材：试写开篇来定夺

当然，还有人喜欢看的类型很多，喜欢看大总裁，也喜欢看古言宅斗，偶尔还看看末世，然后开文的时候就开始纠结了，都想写，但不知道要开哪个。

这种时候可以先想象一下每个题材的核心情节，毕竟一般大家想要开新书也不是随便就想开，脑子里八成已经有个大概想法了。

先在脑子里想象核心画面，这时候先不要纠结大纲了，对比一下脑子里想象出来的画面，哪个更震撼，哪个更让你热血沸腾，迫不及待，就先开这个。

如果想象不来，那就先写一下，每个类型写一千字，感受一下自己写的时候是否是文思如泉涌。如果连高光情节都写起来很难受，那这个类型你可能不太能驾驭，就先放一放吧。

# 三、情节如何环环相扣吸引读者？

故事情节如何环环相扣，这是一本小说的基础，底层逻辑就是因果关系。

主角做这件事是为了什么而做，为了达成什么目的；又或者因为主角做了什么，导致他们面临一个怎样的结果。

### 3.1 小说的底层逻辑：构建因果关系

很多新人写文不太会注意小说的因果关系，反而是用“然后”关系来推进的，这样就很容易流水账。

比如说：我打开门取了快递，然而又下楼扔了一下垃圾。

这种就是“然后”关系，两个事件之间没有联系。

我打开门取快递，正好看到有人拿着刀站在隔壁门口，我慌忙关上门，没多久，我家门口响起了一阵敲门声。

这种就是因果关系，“我”开门看到了对面持刀，所以持刀人找上了“我”家。

每个情节都应是前因导致的必然结果，同时制造新的障碍，为了解决这个新的障碍又不得不做一件什么事，这样子推动剧情向不可预测方向发展。

因果关系是一篇小说最底层的逻辑链，小说也是由无数逻辑链组成的。

写一个故事情节的时候，多问问自己为什么，我为什么要这么写，这个故事写出来对文章情节有什么推进作用？它会引起一个怎样的悬念，或者可以填掉什么时候挖的坑。

### 3.2 用人物欲望推进情节

在因果关系的基础上，用人物欲望来驱动情节，主角的核心目标比如：复仇、求生、爱情。

每个场景中，人物行动都应有 “我想要怎样，所以必须怎样的逻辑。” 的内在逻辑。

因为要为全家复仇，所以现在在仇人家忍辱负重。因为爱情，所以在火灾的时候，冲进家里救自己妻子。

### 3.3 埋设伏笔与回收

在这些整体逻辑的基础上，可以再加上一些小巧思，就是我们俗称的伏笔。

比如主角在前期救过一个小孩，后期遇上麻烦的时候，这个小孩带着家族大佬强势来救她。

在各种逻辑链的交织下，小说就一定会环环相扣，情节一波未平一波又起。

### 3.4 所有逻辑链服务主线

但是在写这些逻辑链的时候，很多新人又会陷入另外一个误区，结构太过涣散。被某一个逻辑链拉偏了，脱离主线。

比如小说的主线是女主重生复仇，然后女主习武是为了复仇，却又在某些情节的干预下跑去破案了，破案几十章。破案几章没钱花了，又跑去做生意等等，几十万字写下来，可能读者连女主一开始要干什么都忘了。

所有的逻辑链最终都要汇聚在一起，这么多逻辑链都是为了支撑一条主逻辑，主角为了达到怎样的目的，所以做了这一切。

去破案，那每一个案子都要发现一点跟复仇相关的因素，最终案子聚焦在仇人身上，女主彻底扳倒仇人，大获全胜，文章大结局。

情节环环相扣就好像一个钩子勾着读者，让读者好奇接下来到底会发生什么，这样才能最大程度地留下读者，不会半途弃文。
