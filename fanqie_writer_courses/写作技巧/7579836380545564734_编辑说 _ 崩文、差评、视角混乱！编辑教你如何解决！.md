# 一、小说写着写着突然崩文了怎么办？

我们都知道，一本优秀的小说刚开始写作时都相对艰难，作者们在小心翼翼地磨过开篇的黄金三百字、磨过黄金三章、磨到八万字上推荐，整个都非常顺利地写了一二十万字，结果在这时候突然一个换地图垮掉，前面辛辛苦苦积累的读者流失了大半，整本书的数据也就这么崩掉了。今天我来跟大家分享剧情崩文该怎么办。

## 1.1 崩文的常见原因

①前期的大悬念被揭晓了，而后面的悬念没有跟上，导致了读者流失；

②换地图不够丝滑，割裂感比较严重，读者出戏了，就会换本书看。

这两个原因基本都是因为悬念交替做得不好，没有提前做好两个接力棒之间的交替，等到发现崩文以后想去弥补这个问题，但事到跟前发现并不好改。

## 2.1 提前预防的3个建议

①悬念需提前交替铺垫：

小说在第1个悬念揭晓之前，就要考虑铺垫好第2个悬念，注意这里指的提前铺垫，并不是指在第1个悬念快结束后才铺。

因为你的书名简介和前期铺垫的重点都放在了第1个悬念上，等快结束的时候，读者的注意力其实都还集中在了你第1个悬念上面，此时再去铺第2个悬念的点，已经有点晚了。

这里更建议作者提前未雨绸缪，就比如说你在前期写第1个悬念的同时，就要在穿插着跟第2个悬念相关的人或者事物，在前期除了主悬念以外，还要增加另一条或者两条线来勾住读者。

两三根绳子肯定比一根绳子抓得稳吧？

例如：

在悬念1的故事中，会穿插悬念2的部分故事，穿插悬念3里人物的露面或者提及；

等写到悬念2的时候，穿插悬念3的部分故事，给悬念4的露面或者提及。

一层层往下铺，让每个悬念故事线的钩子都至少保持在1.75个及以上，丝滑切换。

②换地图应聚焦故事而非背景：

就是在换地图的时候，我们更倾向于把重点放在这个故事的本身，而不是在于背景上。

例如：

主角是需要去一个地方，去拿到什么东西，他去拿东西就直接写拿东西就好了。

不要过度描写他去路上的心理历程，以及去到那以后那边的环境描写。

不要让读者明显察觉到作者在换地图，要让他在不知不觉中跟主角完成了多少事，而不是走过了多少地方，我们要根据人物或者剧情的冲突走，弱化了换地图的问题。

③新人可考虑暂不换地图：

对于新人作者来讲，如果掌握不好换地图的技巧，那就用最笨也最简单的办法，尽量不要换地图。

我们可以把故事线集中在一个地方：

例如：

写校园文就是全本在学校，不进入职场；

写宅斗文就全本在宅院背景、不进战场甚至朝堂。

实在非得换地图的情况下尽量往后拖，字数越长后读者的弃文概率越低，而且就算真崩文也快完结了，能将损失降到最小。

总之，大家可以根据自身的笔力，来保证自己在能力范围内创作，尽量多留住读者。

# 二、如何处理有道理的读者差评？

之所以跟大家分享这个案例，不是为了安抚大家被读者刺激到的心灵，而是教大家如何利用好读者的评论，来帮助自己写文创作。

## 2.1 核心心态

事实上我们每个人都不希望被批评，更不希望读者批评自己的同时、把自己精心创作的小说打个一星差评，这种是真的很难受。

但是呢，批评其实也确实会是成长比较快的一种方式。

例如：

你的主角有说出不符合当下主流的三观言论，会让无数读者愤怒过后怒打一堆差评。很多作者这时候除了都会生气之外，大多会有两个反应。

①跟读者激情开麦，两人把差评顶到书籍最首页，让差评被更多读者看到，影响更差；

②是反思自己：这个剧情是不是有问题，我要不要改文呢？

关于要不要改文这个点，我也给大家分享一个真实案例吧：

我有作者说：“编辑，我第1本书小有成绩，但读者骂我的特别多。于是我在写第2本书的时候，把我第1本书里面所有被读者骂到的毒点，全都改掉了。然后……扑街得非常惨。这是为什么呢？我明明那么听她们的话了，为什么反而没有第1本书的成绩好呢？

首先我觉得可以稍微跟大家说点不太好听的建议，虽然作者跟读者的关系是可以处得很好，但这两种关系还是有一点对抗的。这种对抗不是指两方人是敌人，而是指读者想要什么，作者不能立马给你的那种对抗。

例如：

一个虐心情感拉扯的文，男女主刚拉扯了10来万字，水都没烧开呢，所有的读者评论都喊着让他们两个立马在一起/立马揭开所有误会/立马分手。

这个时期的读者情绪期待值很高，就是在等这个大悬念的揭晓。

但如果你听到读者们的集体呼唤或者来了多个恶评威胁，立马让你本该30万字写完的剧情10万字就给他们全部看到了，那么会有一个结果：整体的节奏都被他们打烂了。

这也就涉及前面我们讲的崩文问题，你的第1个大悬念被影响的很快结束，第2个悬念点还没铺好，读者心满意足地看完、一点都不停顿地就奔向了别的作者怀抱。

所以大家不要被读者们的花言巧语骗了，等她们爽完就跑，哭的人只有作者。

## 2.2 如何判断与处理差评

那怎么判断读者的评论要不要听？哪些是合理的，哪些是不合理的呢？

首先，那些我们急于更新而没细查的常识性错误，这种小的读者建议可以参考改正。

但是剧情向、关乎到你整体的主线设定和故事节奏的，就要慎重考虑。

这里有个最客观的表现就是数据，你可以根据你作品的追更情况，来决定是否要改文。

因为如果你的文真写崩了，你的数据以及收益上是会有反馈的，这个作者是能察觉到的。很多读者包括尤其是虐文读者，他就是喜欢边骂边看小说，甚至有的书整本几千个评论里全是差评，没有好评，但作品就是有很多读者追。

所以元宝也希望大家站在作者的角度：可以稍微冷静、客观地看待读者的评论，能帮到我们的地方，我们可以根据评论去修改一些常识性逻辑性的问题，不能帮忙甚至还想干涉作者主线剧情逻辑的——这种我们可以假装没看见。

# 三、视角切换在小说中是好是坏？

## 3.1 视角割裂：

其实我们很多作者看电视剧比看小说要多，或者说看电视剧的时候没有细细研究过电视剧与小说的不同，像这种情况在写小说时，就会有一个非常明显的问题，就是上帝视角。

我们在看电视剧的时候，用上帝视角看多个主角或者配角，会觉得他们的人设非常的丰富，场景非常宏大。

但在看小说的时候，如果视角不是在主角身上，而是在各个角色身上跳来跳去、跳来跳去，那读者的代入感就会非常的差。

他写出来的感觉更像是换了一个又一个的小地图，那种割裂感会一次次降低小说的代入感。

## 3.2 具体建议：

· 前期聚焦主角视角：

所以我们在写小说的时候，尤其是开篇前期几万字，尽量是围绕着主角一个人以她的视角来展开。其他人发生的事情，不管是一笔带过还是说听别人提怎么样，都要保证主角一定是在场的，尽量少切或者不切其他人的视角。

· 后期可少量添加：

等到你的字数长一点，你的读者的代入感能够进入深度的阅读体验以后，可以再稍微加一点别的视角的描写，控制在几百字的篇幅较为合适，尽量不要大篇幅的来讲。

因为我们也有出现过火文突然开始在配角地图上几章几章的讲起来，这种就很考验配角的设定，如果他的设定没有比主角更出彩的话，那读者可能会在这部分没有主角的部分，就会发生弃文的情况。

关于主角视角，可以给推荐大家看一本书，丛月的《首辅家的锦鲤妻》前几章都是女主视角，一口气读下来非常流畅。

总结就是：上帝视角展现出的东西确实非常有画面感，但是在还没有代入这本小说的情况下，想把读者深度代入进去还是挺难的，小说跟电视剧的呈现方式在这里会有明显区别。
