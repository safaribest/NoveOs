# 一、开新书选什么品类？

## 1.1 给老作者的建议

· 深耕擅长题材，微创新设定

首先，编辑会更建议老作者就深耕自己更擅长的题材。

假如你在某个分类的成绩还算不错，那么你就可以一直专心研究这个分类，在这个分类里，把各种设定都写个遍。

例如：

年代文，你第1本写年代替嫁，第2本你写年代姐妹同嫁，第3本写年代重生复仇等等。

之所以这样建议，是因为你在这个分类是非常熟悉的，你知道这个分类的细节设定、写文风格以及读者群体，在这些都了解的基础上去做新书的设定，它会比换分类的成神率更高。

因为每个类型的读者群体并是不一样的：

例如：

像年代的读者受众，可能是中年读者居多；

但你如果改写快穿系统之类的，他的年龄可能就是学生读者居多。

读者年龄不同，三观不同，同样一件事情在他们看来的褒贬不同。

例如：

说一个很火的疯批男主设定，在学生看来会觉得非常带感、刺激，但在中年读者看来，就会想报警。

所以我们更建议作者在自己擅长的领域，新书继续看这种题材，研究透这个分类的读者的三观喜好，想创新可以从换设定上入手，而不是换题材。

## 1.2 给新作者的建议

· 从阅读爱好入手，选熟悉题材

而如果你是写了好几本书，但成绩都不符合预期，那该怎么选择分类呢？

市场上几十上百个分类，看得人眼花缭乱。

当你不知道哪个适合你自己时，我建议可以选择你看得最多的那个类型。

例如：

我之前有一个作者从来没有写过文，她写第1本书就挣了二十万。这种一本就出成绩的作者挺少见的，我就找她聊了聊，结果她告诉我说：

“我确实是没有写过文，但是我写的这个分类，不管是咱们平台还是别的平台，只要是榜单上的书，我全都看过。这个分类它的写法、它的人设、剧情我都了解，我知道她们是怎么写的。”

我当时听完非常震撼，到现在过了好几年还记得这句话。

这位作者对这个分类的了解程度，甚至于比这个分类的很多作者都还要熟悉，所以她作为一个读者转作者第1本书去写这个类型，就很容易起来。

所以我们也建议大家，如果不知道自己想写什么类型的，就从自己的爱好入手，选熟悉的题材写，要不然不熟悉的类型也容易卡文。

# 二、精心构思的第二本，不如第一本火？

## 2.1 核心误区

· 努力不等于成功，过于追求“精品感”

辟谣：其实并不是因为努力了反而没有好收获，这个是对努力最大的误解。

很多作者在第1本书小有成绩之后，肯定是希望第2本书比第1本更好，所以才会做这么完善的设定和人设，希望这个故事能够比第1本书成绩更好。

但实际上，由于作者大大们太过于注重精品感了，就是过于追求要写一个非常精品的故事，想让他大火，想让他改影视，想得太深入，反而忽略了最基本的网文的入门写法。

## 2.2 关键点

· 注重“前台故事”，吸引读者阅读

《宝莲灯》的编剧九年说过：“很多作者太注重写后台故事，反而忽略了前台故事。”对这句话我深有感触，分享给大家。

我理解这个前台故事就是指：在最开始你吸引读者阅读进去看这个文的点。

例如：

开篇是一个攻略文，男女主是一个对抗的冲突设定，女主拼命地想攻略男主，而男主拼命地想杀女主，这是一个吸引人的前台故事。

而那些男女主身后的庞大世界观以及两个人人设更深层次的挖掘，这都属于后台故事。

很多作者在写第2本书的时候，就是忽略了前台故事要接地气、要好看，只写了后台故事的高大上，这样读者其实也很难发掘后面的好，因为酒香也怕巷子深。

所以大家一定要记得：是在你之前火文的基础上精益求精，而不是完全抛弃以前的成功经验。

# 三、套路文能火，创新文却扑了？

## 3.1 市场现实

· 读者群体多元，“老套路”对新读者仍是“新设定”

这个问题有人站队套路为王，有人站队创新为王，而今天我想讲的是：套路和创新并不是绝对的对与错。

市场上的火文未必就是老套的，一味追求创新的开篇也未必是好看的。

最近几年，网络文学被免费市场开拓得越来越大，导致各个年龄段的读者涌进来了，小至几岁大至七八十岁都有。而这些各个年龄段的用户并非都是老读者，有人六十岁才开始看网文，而那些我们很常见也觉得烂大街的设定，对他们来说可能就是非常新颖的设定，因为他们没看过小说。

也有一句话说的很好：“没有人永远十八岁，但永远有人18岁着”，换到小说里面就是这种情况，我们会不断的有新用户进来去看一些比较套路的小说。

## 3.2 创新建议

·“新瓶装旧酒”，在套路基础上微创新

但是，以上说法也并不代表说纯套路的文就一定能火。因为写套路文的作者太多了，在不断推陈出新的网络文学中，能在纯套路文杀出血路的大多是笔力极强的老作者。

从编辑看到的作者们作品数据来看，更多的老作者还是被后浪拍在了沙滩上，新作者拼笔力拼不过老作者，就会在设定上推陈出新，赶超前人。

因为不管套路用的有多好，读者自始至终都是想看一个吸引人的好故事。

编辑建议作者走的创新也不是纯创新方向，而是在套路的基础上去微创新。因为套路是网文无数代前辈总结出来的非常惊艳且实用的写法技巧，我们在创新时找一个新颖的切入点，再用套路好好包装下，那它就是一个非常好看的创新小说，这个叫做“新瓶装旧酒”。

而抛掉所有套路的纯创新，所有的东西都没有用套路的写法，所有的东西都是纯创新的，也就是说前面几十年前辈们总结出来的几百上千个经验你全部都不要，全部都要去写自己的纯创新，这个还蛮难的。

我觉得可能会有这样的天才作家出现，但是这样的概率非常小。还是那句话，因为编辑可以很直观地看到作者们创新的结果概率。

## 3.3 创新比例

· 循序渐进，避免影响写作道心

虽然编辑也很希望说未来我们能出现一个这样的天才，把前面几十年所有网文前辈总结的套路都打败、都超越，但这条路很难走。

编辑还是希望作者能够走一条尽量宽敞的道路，你先写一本爆火的文。等你火了以后，你的笔力也会非常强，那到时候你想去大创新的成功概率也能更高一些。

所以在这里也可以分享一个数字，创新的比例我们仍然建议不要太大。

例如：

你火了的这本书创新比例是：在90%套路的基础上有10%的创新，那么你在火了成为一个知名的大神以后，你想再写一些更新颖、更新颖的东西，可以提升到20%或者是做一个稍微新颖的世界观。

但你在全新世界观里面的人设设计手法，和人物发生的一些人物冲突，这些都还建议用你之前积累的成功技巧来写。

就这样，步子别迈太大，稳扎稳打地一点点创新，以免创新太狠被差成绩影响了写作道心。

总结就是：不要极端地觉得套路一定不好，也不要极端觉得创新一定好。

# 四、怎么去研究学习火文？

温馨提示：这只是让大家去学习火文套路和写作方法，严禁抄袭哦！

例如：

很多作者会去研究一个火文，看完后跟着去写一个相似的新书。

而自己的文里写的主线和走向跟对方一致，为了跟对方有区分，别人写哥哥她写的是弟弟，看似不一致，但实际上人物关系跟框架是一致的；

就很容易造成一个结果就是高仿或者融梗，哪怕这并非作者本意。

作者们在学习火文的同时，我觉得有三个点需要重点注意：

## 4.1 避免模仿

看文的时候不建议你只研究一个文，因为不管你是有意的还是无意的，真的会写出来跟人家很像。

例如：

你要研究一个后悔流的小说，那么你就找两三本后悔流，来看看他们这三本书里的共同点都有什么。

如果这三本书里都有的元素，那么你也是可以写的，这叫火文的共通点，也叫套路设定；

如果你只看其中一本书，按照它的主线剧情整个写下来，让整个故事线都连贯的相似，这种就属于连环撞梗，也就是抄袭。

## 4.2 学习本质

作者们在研究火文的时候，不要只看别人的人设和故事设定具体讲了什么内容，而是要去学习他们最本质的内容。

例如：

其他人做这种类型的人设为什么会讨喜？

或者说是他们的故事选材是从拉仇恨还是温馨升级流？

亦或者是做期待感等方式等等。

学别人学火文最本质的特质，而不是去学习别人的剧情，这个是非常需要注意的。

## 4.3 完整阅读

不要为了图省事只看开篇，一部火文的精髓不会只在开篇里，很多作者埋的伏笔、写法用意，都在后面慢慢展开。

而在没品明白的情况下，学出来的东西很容易浮于表面，亦或者后期不知道怎么把控全文。

所以编辑会更建议大家尽量看个完整的百万字长篇小说，这比只看十个开篇更有用，它会大大增强你的长篇把控能力，降低崩文率。
