# 一、对写出来的内容不满意？

通过你的叙述，你给我的感觉就是你已经掌握了一定的写作技巧和方法，然后信心满满动笔去写，结果写出来的东西却不是自己想要的，和自己想要的那个水平差了很多。

因为我没有看过你写的文，所以我不知道我形容的是不是准确的，所以我只是从我的角度来分析一下你现在遇到的问题。

### 1.1 写作是探索，而非解题

其实你这个情况，是把自己给困住了，这是一种什么感觉呢，就是你太把网文写作这件事，当成一道数学题，或是当成一门需要考试的科目去做了。

说白了你是带着答案去找问题的，你把网文写作这件事当成了一道题，一道有了题目就必须有正确答案的数学题，你以为你按着步骤和公式，一步一步做下去就能把答案给解出来，其实不是的。

你把这个先后顺序给搞反了，写小说不是带着答案去找问题的，而是带着问题去找答案。

### 1.2 大纲与人设是辅助，而非枷锁

说白了，你上面说的那些完整的大纲、伏笔、反转、人物小传这些东西，其实都是辅助工具，这些辅助工具是帮助你开拓思路和捋顺故事逻辑用的，就好像婴儿要学走路，所以需要一辆代步车，而不是因为你在淘宝买了代步车，然后不论你是否练习，你直接就会走路了，这个逻辑是不对的。

我之前的课程就说过，每一部网文小说都是一盘番茄炒蛋，现在我把炒蛋的步骤方法告诉给大家了，但是每个人炒出来的这盘菜，味道都是不同的，有的人盐放多了，有的人糖放多了，有的人甚至忘记了放鸡蛋，变成了番茄炒番茄。

所以我要说的是，网文写作，是没有正确答案的，它不是一道数学题，也不是一项维修技术，不是你学会了维修步骤，就能学会这门技术，那些所谓的技术技巧，它仅仅只是一种辅助工具，是用来帮助大家开拓思路的。

### 1.3 调整心态，感受到阻力正是进步的标志

你现在遇到的这个问题情况，其实在写作过程中，是非常常见的，就像你说的，你把所有的写作技巧都用进去了，你字字句句地去斟酌，可结果连差强人意都做不到，然后你就灰心了，不知道怎么办了。

我这里是要鼓励你一下的，你不要有灰心的心态，这种情况不奇怪的，反而，在你感受到阻力的时候，正是表明你要进步了，你是没问题的。

### 1.4 过度准备压抑灵感，关键在于坚持和勇气

那我说一下你现在出现的问题是什么，你就是准备的太多了，人一旦在做某件事之前，就不说是写网文，做其他事也一样，如果准备的太多太多的话，是很容易把自己给束缚住的。

你说你大纲准备的特别好，人物小传准备的特别好，那我问你，你在写作的时候，是不是时常会想着说，诶，我这个部分的大纲是怎么写的来着？诶，我那个人物是怎么设定的来着？

你看，当你有这种思路的时候，就说明你被困住了，你被困在了你之前的所有准备之中，你反而没有灵光一现的写作灵感了。

而且我之前的课程就讲过，厉害的网文作者，最后写的就是一种“感觉”，这个感觉是不好形容的，它是一种能把整个故事顺下来的灵感，就那么一根线，一根很小的故事主线，然后开枝散叶，变成一个很庞大的故事。

简而言之，就是你准备的太多了，身上背负的东西太沉了，然后就导致你迈不开脚，跑不起来了，导致你原本能在写作旅途中大胆畅想的东西，都被你的大纲和人物小传给压抑住了。

所以你一定要记住，大纲和人物小传，它只是个辅助你写作的工具，那东西就是个帮助新手作者能够快速上手的工具，就跟那个婴儿代步车是一样的，最终婴儿能不能够学会走路，你能说是代步车的功劳吗？不是的，我们能不能学会走路，最后靠得是坚持的练习和敢于摔跤的勇气啊！

你总不能说，我没有代步车，我这辈子就学不会走路了是不是？咱不能让大纲和人物小传给束缚住了，这样就得不偿失了。

那放在网文写作这件事上，我们能不能成功的关键，是在于我们能不能每天坚持写作，以及，有没有接受自己无数次扑街的勇气啊。

这个道理都是一样的，是万变不离其宗的。

### 1.5 轻装上阵，三招摆脱束缚

那我们最后说一下，怎么解决这个问题，第一，我希望你大胆一点，如果发现人物小传和大纲束缚了自己，那我们直接把这两样东西给扔掉，推翻，你就大胆往下写，哪怕逻辑不顺了，跟你最开始的设想背道而驰了，没关系，大胆写，键盘在你的手里，如果你写故事都不能由着自己的心，那还有什么事能由着我们自己呢？

第二个就是，我觉得你的素材积累和储备可能不太充足，建议你多去看看电影看看书，这样是可以缓解自己的写作焦虑的。

第三个就是，专门针对你的办法，你以后写作，不要上来就开始准备故事大纲和人物小传了。

从今天开始，你不要准备人物小传和故事大纲，你先去动笔写故事，就是你脑袋里构思出你想写的短篇幅的故事开头，然后你先拿出电脑去写，然后，在你往下写的过程中，你一点一点地，去创建你的故事大纲和人物小传。

就是你要先写出故事开头，然后再在创作的过程中，去梳理你的大纲和人物小传，你要先有故事，并且要先写出来，然后再有大纲和人物小传，以后就坚守这个写作顺序，你的问题就能解决掉。

如果其他人也遇到了这个问题，就按着我说的去做就行了。

我们是创造故事，创造宇宙的人，我们可不能被自己给束缚住了。

# 二、人物总是扁平化？

这里我强调一点，优秀的人设是能够推动剧情发展的，及格的人设是为剧情服务的，不合格的人设是拖垮整篇文章的。

你自己也说了，写着写着，把主角写成了工具人，只为剧情服务了，这种就是刚过及格线的人设，也不是太差，但优秀肯定谈不上，主角被写成了扁平化，没了自己的特色。

### 2.1 核心：缺乏撞击感与背景故事

其实就是人物设定不够有撞击感，不够丰满，被剧情给推着走了，显得特别的被动。其实这个时候是可以从人设的背景故事去下手的，说白了就是丰满你的人物设定，当一个人物有了属于他自己的背后的故事，那这个人物就很容易在你的故事里立住脚。

好的人设肯定是能够推动剧情发展的，因为他自己就有足够多的故事和力量。

### 2.2 两大症结与解决途径

人物性格很大众，一是你的人物设定真的很普通，二就是你的剧情很普通，反正就这两种原因。

人物设定很普通，那是你的人物素材积累不够。

剧情普通，那就是你设计剧情的功底不足。

因为我没有看过你写的小说，所以我没办法具体问题具体分析，但大概就是这两个方向，要做的就是多做素材积累，多看书看电影，这个准没错。

### 2.3 根据后台数据进行优化

首先不要担心自己写了几本之后会毫无长进，这种担心属于杞人忧天，没有必要，你就闷头写就行了，只有写出几本书以后，你才能看到自己的后台数据，看到番茄的读完率，跟读率这些东西，这些数据能比较能直观地反应出你这本书的问题所在的，开篇够不够吸引人，后续黏性强不强，都是一眼就能看出来的，然后你对症下药去修改就行了。

然后，你去跟榜单上同类型的优秀作品做对标，你看看人家好在哪里，你差在哪里，这种方法能让你快速找到自己的问题所在，是能快速进步的。

好作品的优秀之处都是大同小异的，即便你换了新的题材，你学到的东西也是能用在你的新题材里的，不存在说无从比较这个情况，所以当务之急，是去复盘和分析，加油！

# 三、核心创作技巧解析

### 3.1 建立写作心态

· 克服数据焦虑的关键：

初期不以赚钱为目的，只以写作乐趣为目的，自然就会有好心态。

· 应对灵感枯竭的方法：

高阶作者不靠灵感写作，职业作者如果靠着灵感写作，会被饿死的。灵感枯竭只针对初阶作者，反而需要疯狂练习和疯狂的素材积累，没有灵感要去找，而不是单纯的等。

### 3.2 创作无框，精彩为王

· 让对话描写更具感染力：

故事来源于生活，你怎么想怎么做，或是你身边的人怎么想怎么做，你的故事就可以朝着这个方向去设置。

你在被另一半欺骗背叛的时候，你会做什么说什么？你的朋友会做什么说什么？你和你男朋友一起买了一张彩票，中了五百万，这个时候，你会做什么说什么？你男朋友又会做什么说什么？是开心庆祝？还是杀了对方独吞？

· 关于创作流派的见解：

感情流的文章，就是感情主导，剧情偏弱。反之，剧情流的文章，就是剧情主导，感情偏弱。你不用刻意给自己划分剧情流还是感情流，也不用刻意去寻找平衡点，你只要写精彩了，什么流都行，没有硬性要求，新人作者最怕的就是给自己增添莫名其妙的条条框框，没必要。
