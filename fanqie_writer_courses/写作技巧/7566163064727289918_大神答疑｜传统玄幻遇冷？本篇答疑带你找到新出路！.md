# 一、传统玄幻是不是不行了？

首先，我声明一点，以下都是我个人观点。

## 1.1 问题根源：同质化严重，缺乏新鲜感

我个人感觉，传统玄幻不是不行了，而是存量太多了，市面上大同小异的长篇玄幻小说太多太多了，五百万字一千万字以上比比皆是，而且很多小说，都曾经红极一时。

你再去写一本类似的传统玄幻，字数少，根本就没有任何优势。

而是一些作者，写新书，又摆脱不了传统玄幻的一些框架：

比如检测场，家族势力争夺，宗门试炼，秘境夺宝，拍卖会，比武等等。

任何这些框架之中，又延伸出许多常见的套路，比如退婚，丹田被废，逐出师门等等套路剧情。

这种常见的套路框架，几乎在绝大多数传统玄幻都能看到，包括我写的小说，一样会写这些类似套路框架。

很多喜欢看传统玄幻的读者，他们可能看的传统玄幻小说，比一些作者还要多，如果你写的传统玄幻，写这些类似套路框架，和其他玄幻小说大同小异，给他们带来不了新鲜感，甚至他们都能猜到你要写的剧情，哪怕你写的再爽，但在那些读者的观感下，爽感会大跌的，那成绩自然就起不来。

除非，看你小说的读者，全部是完全没有看过小说的萌新读者。

这就好像喝可乐一样，最爽的永远是第一口。

## 1.2 破局方法：旧酒装新瓶，进行“微创新”

那如何摆脱这些呢？

我个人觉得有两点：

第一点，就是摆脱这些套路和框架，写脱离这些套路框架的剧情，但这对于很多新作者而言，是一件吃力不讨好的事情，我们就是看那些套路小说长大的，你写玄幻小说，脑海里肯定会浮现，各种秘境夺宝，拍卖会，宗门试炼等等剧情。

甚至，传统玄幻小说的套路，在网上都被编成了顺口溜。

你写任何剧情，好像都脱离不了那些套路一般。

不说新作者，哪怕老作者，写传统玄幻，都很难摆脱这些套路和框架，写几本小说，都有可能是围绕这些套路框架写的，唯一不同的就是剧情不一样。

既然摆脱不了，我们要做的，只能旧酒装新瓶，在熟悉套路框架之中，做出一些改变。

这就是我所说的第二点，在设定上做一些改变，比如人设上的改变，金手指上的改变，甚至世界观上的改变，做一些反套路，微创新。

### ①人设创新

比如我《万古第一废材》里前面的要写一个剧情，一个师姐要组四人小队出任务，斩杀妖兽。

这种在其他玄幻小说里，常见的不能在常见的剧情，如果依旧按照老路子写，肯定没什么人看。

甚至很多读者会觉得没什么意思，做任务的剧情，完全就是在拖节奏。

当初，我构思这个剧情，也觉得会拖节奏，但我必须要写，是要铺垫一个东西，必须写这个剧情。

我构思很久，才想到，围绕主角是废材人设去写。

所以写了秦玉师姐为了磨砺自己，故意招三个废材组队做任务，而主角废材人设，用一个“比废”的剧情，成功加入秦玉师姐队伍。

不仅仅提高了剧情的趣味性，也利用信息差，增强了读者的期待感。

再比如，列举几个以废材人设的剧情，比如我后期陀舍洞府的剧情，设定，只有废材才能得到陀舍洞府传承，

这让一个其他小说里，人人瞧不上的废材，反而成了优势。

### ②金手指创新

至于金手指上的创新，如果能做好一个创新的金手指，那也能给读者带来新鲜感，网文二十多年，从最早的没有金手指，到后面，戒指老爷爷，然后，各种逆天功法当做金手指，到后面，各种法宝当做金手指。

设定金手指，到现在很难很难了，大同小异的金手指已经层出不穷了，我记得几年前，流行的金手指，就是吞噬一类的，当初一些网站甚至主收吞噬类的小说，到后面，各种宝塔类的金手指出来了，到现在，番茄小说里的传统玄幻，好像几乎都是宝塔类的金手指，开启一层一层，能得到各种至宝。

我个人觉得，设定金手指，一定要遵守一个原则，给读者一种无敌的感觉，但又有一定的限制，否则写的太无敌了，最终的结果，肯定会导致剧情崩盘，情绪崩盘，关于情绪崩盘，在后面一个问题里，我会详细说明。

就拿宝塔类的金手指举例，每一层都藏着一种至宝，开启一层，就能给主角带来巨大提升，这就会让读者期待，另外的那些至宝，这种就类似于玩游戏，七天奖励一样，登录七天领取奖励，越到后面，奖励越好，就给读者一种期待感。

但现在，宝塔类的金手指已经太多了，同样带来不了什么新鲜感了，新书想要出成绩，我个人觉得，还是要再构思一些创新的金手指。

### ③世界观创新

至于世界观的创新，世界观包括，修炼体系，世界规则，修炼体系，我们耳熟能详的就是斗破苍穹的斗气修炼体系，斗罗大陆里的武魂修炼体系等等，想要在世界观上做创新很难，世界观的创新，意味着给读者带来陌生感。

我个人认为，修炼体系上，新人做人尽量不要搞创新，让读者越容易代入越好，世界运行规则上，如果能让读者代入，引起读者情绪共鸣，我觉得就成功了。

比如我《万古第一废材》里的，血脉分三六九等，只有天赋强的能修炼到最强，普通血脉没有修炼资格，就容易让读者情绪带入。

最后总结一下，写传统玄幻，想要在当下市场出成绩，我个人觉得一些要有改变，人设上的改变，金手指上的改变，世界观上的改变，唯有这些改变了，写那些老套框架的剧情，才更容易出成绩。

# 二、剧情和文笔哪个更重要？

网络小说里，当然是小说剧情重要，一个能有效传递情绪、让读者有代入感和追读动力的故事，远比辞藻华丽更重要。

文笔真的重要吗？

文笔恰恰是最不重要的一项，文笔是为剧情服务的工具，而非目的，不要让文笔不好成为动笔的障碍，句子通顺、能把故事讲清楚才是核心，文笔这种东西，可以通过长期的写作，慢慢提升起来，所以新人作者，不要担心自己文笔差，写的不怎么样，你只要能通过自己的文字，把自己心目中的剧情写出来就可以。

# 三、如何选择题材并进行创作准备？

## 3.1 如何选择题材

我觉得选择题材，还是要看自己喜欢看什么题材的小说，你喜欢看小白文，那就去写小白文，你喜欢看群像文，那就去写群像文，你喜欢看脑洞文，就去写脑洞文，你如果自己都接受不了一个题材，仅仅因为那个题材赚钱，就硬着头皮写，肯定成功不了，每个题材都有自己的受众，在自己喜欢的题材深耕，肯定有火的一天。

至于分析受众，我觉得如果不是写新媒体小说，没必要分析受众，只要写的好看，能让自己爽起来，肯定都有人看，还是回归上面选择题材方面，选择自己适合自己的题材去写。

当然，如果你累积了足够的写作经验，可以去尝试自己不擅长的题材，受众比较广的题材。

## 3.2 创作准备

关于金手指，世界观架构，我觉得这不是想要开书，就突然能构思的，这是一个累计的过程，可能看部电影，看个动漫，看一本小说，玩一个游戏，就灵光乍现，想到了一个金手指，或者一些好的世界观，将你觉得有趣的金手指，世界观记录下来，常年累计，量变肯定会产生质变，开书的时候翻一翻，肯定会带来意想不到的效果。

至于风趣幽默的句子，我觉得这和个人性格有关，如果写不来，不用勉强，当然，有些小说里，免不了有些搞笑角色，我个人的经验是收集一些抖音里比较搞笑的梗和段子，然后，融入到角色对话和剧情中。

# 四、如何培养写作心态和写作习惯？

## 4.1 应对卡文

我不知道你们眼中的卡文是没有剧情还是什么的，我眼中的卡文，是我可以写，但写出来会不好看，会很平，不是很爽，或者说写出来，会和以前的剧情，有结构上的重复，所以我会停下来，思考一段时间，重新构思剧情，先让自己爽起来，如果实在想不出来，那就硬着头皮下，快速写，有时候写快了，也会突然迸发出很多灵感。

一定要写，再卡也要写，硬着头皮也要写出来，尤其是新人，每一次卡文，对于你们而言，就如同修炼的瓶颈一样，一旦突破，写作上就能得到一次提升。

## 4.2 坚持创作

我觉得创作，首先肯定是因为热爱，只有真正热爱，才能坚持写下去，如果新人一来，就冲着一本成神，赚大钱的目的，年入百万，肯定难以坚持，真正有天赋，能够一本出成绩的还是极少极少数，很多赚钱的作者，真正能够靠稿费养活自己的作者，都是累计了几本书的经验。

我还是新手的时候，写书也是凭借满腔热情，脑海里的素材，只有以前看过的几本小说，也不知道任何写作套路，就是憋着一股劲去写，但读者也被主角那股劲感染到，选择追读下去。

当然，我也不建议关门造车，仅凭自己一腔热血去写，网文终究是商业写作，是商业，那肯定有套路的，有技巧，如果你意识到自己写的烂，那说明你已经进步，边写边学习一些套路，一些写作技巧（番茄作家助手里，就有很多大神作者分享的写作技巧）。

如果可以，坚持写一百万字，我个人觉得，能写出一百万字的小说，无论好坏，都已经算入门了。

# 五、如何写大纲并稳定故事节奏？

## 5.1 大纲怎么写

我是写升级流玄幻的，我个人开书前的大纲，其实很简单：

比如，做一个修炼体系，等级设定，地图设定，势力设定，武器丹药设定，以及前期几个人物设定，男女主设定，主角金手指设定，就没有了。

我大纲里没有任何主线，我不会说，我写这个小说，我要给读者去将一个什么故事，因为长篇玄幻小说，动则三五百万字，甚至更长。

我的主线，甚至我要讲一个什么故事，是根据开篇来的，只有我把开篇写出来，要围绕开篇，确定主角的人设女主人设，金手指等等，再去完善大纲的主线，世界观等等。

那个时候，我可能会确定，我要讲一个怎么样的故事。

但我不会固定主线，很有可能，写着写着，主线就变了，但世界观一般就定死了，但也只会先定死一个小地图世界观，随着剧情发展，再慢慢衍生到大地图世界观。

我觉得，写小说，作者也要和主角一样，是慢慢去探索一个世界，随着主角实力越来越强，越来越大的世界观，也会展现出来。

## 5.2 保障剧情流畅

在写作中不自觉的添加各种说明性的文字，导致节奏被打乱怎么办？

我觉得，这就是没有铺垫好的原因，我个人习惯，要准备好三个爽点剧情：

比如，A B C，三个剧情，A剧情爽完，要进行B剧情，还要铺垫C剧情， 然后写B剧情的时候，我要开始构思D剧情了，因为B剧情爽完，就要进行C剧情，铺垫D剧情，构思E剧情，然后再以此类推。

举个最简单的例子，主角是个炼丹师，他炼制了一枚品相垃圾的丹药（实际上是一枚极品丹药）这是A剧情。

他准备送给公主，当做公主的生辰宴，这是B剧情， 那B剧情当中，肯定会有嘲讽主角的反派，会在生辰宴上嘲讽主角，最后主角的验证丹药，是极品丹药，B剧情结束。

这个反派，肯定会引发一个新剧情，那就是找主角报复，那就是C剧情。

C剧情结束之后，可能还有有D剧情，比如在公主生辰宴上，有大家族天骄，看中了主角的炼丹天赋，想要请主角炼丹，顺便帮主角教训那个反派。

好了，A、B、C 三个剧情，还有后面D剧情，全部都构思好了，而且写出来都会正常的爽文套路剧情，那你就可以在B剧情，把这个反派，和大家族天骄，在B剧情对话中，就引出来，铺垫出来，等到写C剧情的时候，你甚至可以把D剧情构思出来，看是否要铺垫，然后无限循环下去。

只要铺垫好了，一切剧情都会变得很顺畅，如果，你写A剧情完，都没有想到后续B剧情C剧情D剧情怎么发展，那就完全没有铺垫，没有铺垫，那很多信息，都是临时构思出来的，自然会强行用旁白解释很多。

# 六、如何推进情绪？

## 6.1 如何进行有效情绪描写

我们先弄清楚情绪是什么？

情绪就是经过一件事情后，某一个角色受到了冲击，所表现出来的高兴，伤心，生气，愤怒，绝望，羞辱的情绪。

我们在写情绪的时候，一定不要写这件事，而是写这个人表现出来的情绪。

比如，你写，主角家族被屠，主角很绝望，愤怒，这就没什么情绪，你这是在写事情，而不是在写情绪。

你要写，主角看到家族被屠，所表现出来的情绪，比如眼前尸横遍野，血河成河，主角瞳孔猛缩，浑身颤栗起来，瘫倒在地，仿佛浑身力量被抽空一般，随后撕心裂肺大吼：“到底是谁干的？”

你不要写他痛苦绝望，你要让他表现出痛苦绝望。

尤其是新媒体的小说，最注重的就是情绪。而在我个人看来，故事的发展，甚至所谓的节奏快慢，其实也是情绪传递的快慢，我个人认为，小说剧情的推动，就是情绪传递和情绪发泄的过程，情绪传递的快慢，和发泄的快慢，也将影响剧情进展快慢，和节奏的快慢。

而任何小说，我个人认为唯有完成情绪传递，情绪发泄，剧情才不会枯燥。

我们可以举个简单的例子，最经典的退婚剧情，女配角找主角退婚，主角感到羞辱，他就会将这个情绪，要么发泄出去，传递出去，比如，主角身边有父亲，有家人，他的愤怒情绪，就传到他们身上，他们也跟着愤怒了，然后，主角会通过某种方式，发泄这个情绪，比如主角撕毁婚书，扳回面子，但这一行为，就直接将情绪传递到了女配角的身上。

女配角见婚书被撕毁，会表现出生气，她的情绪，又传递到女配角身边的天才身上，他暗中喜欢女配角，跟着愤怒了，然后，他没有传递情绪（可能身边没有其他人），但开始情绪发泄了，亮出自己的身份和地位，蔑视主角，你又屈辱了，愤怒了（情绪更加升级了，因为遭遇到的屈辱更大，比刚才那一波还要强烈）

主角情绪到了，再度传递到身边的人，同时，主角也开始发泄情绪，主角含怒出手，却打不过，被揍了一顿，受尽屈辱（情绪递进），对方更是挑衅，我是要以第一外门弟子的天骄，你这个垃圾，永远没资格和未婚妻在一起。（情绪又一次递进）

然后，你得到金手指（情绪转折），去升级了，然后参加宗门外门考核，强势将那男配角击败（情绪发泄了），但男配角被击败后，也是感到羞辱，他的情绪，传递到在内门弟子的哥哥，哥哥情绪愤怒了，他也选择发泄，出手偷袭你一波，暗中陷害你一波，一步步情绪传递和发泄，便完成一个个剧情的递进。

## 6.2 如何控制情绪节奏

当然，这种情绪递进，还是比较简单的，写到后面，可以考虑，写一些多的情绪转移，让那些转移的情绪，一起爆发出来，可以更爽。

只要一步步情绪递进，主角遭遇到的人物和事，比之前厉害，逼格高，事情大，那就能慢慢递进，但递进的剧情，不能拉的太高了。

比如，你前面递进情绪的都是一些小剧情，可结果，你突然让主角人前显圣，装了一个大逼，你再返回去递进一小剧情的情绪，肯定会蹦。

我们可以看看，有些脑洞玄幻文，开局就是一个大情绪：

比如镇守天渊千年，归来却发现家族被屠，这种类似的剧情，这种通过一个比较宏达的剧情，然后各种情绪传递，甚至多个情绪传递，然后最后主角直接发泄出来。

但这个情绪等级，已经达到了很高的程度，这样写，你后续想要继续完成情绪转移，情绪发泄就很难了。

# 七、如何写好战斗场面？

## 7.1 基础：积累画面感

关于如何写打斗，最重要的还是画面感，如果不会写，其实可以去刷一些动漫，然后，你站在主角视角，学着将这场战斗写出来，多看一些战斗画面，脑海里就能积累素材了，或者就是去学一些大神的打斗，模仿他们的打斗手法。

## 7.2 核心：明确战斗目的

至于如何写好战斗！

首先，你要明白一点，战斗的目的是什么。至于战斗的原因还是和上面的一个问题有关，情绪发泄，任何战斗的爆发，都是情绪发泄。

但战斗的目的，却有很多种，如果你能理清自己战斗的目的，那写出来的战斗，就会好看很多，而且不会水文字。

我们可以举一个简单的例子，比如玄幻里最常见的英雄救美剧情，男主，看到一个美女被恶徒侵犯。

男主看到了，愤怒了，他要做到情绪发泄，然后就要出手，战斗，击败恶徒，救下美女。

但你写这个剧情，这场战斗，目的是什么？如果没有任何目的，就是为了救下美女，得到美女一声感谢，然后事了拂衣去，深藏功与名，那这战斗，你写的再漂亮，那也毫无意义，对于读者而言，那就是水文字。

以前，我经常看到一些剧情，反派无脑嘲讽男主，被男主打脸，然后，就没有然后了，毫无目的战斗。

那什么是有目的的战斗？

目的，依旧是传递情绪！

比如，我写这场战斗，目的是得到美女感激，然后给你报酬，这场战斗，也算有目的，再比如，你写这场战斗，依旧是传递情绪，恶徒愤怒了，他打不赢你，传递情绪给自己的老大，这个老大，对于你接下来的剧情，有推动作用。

一般，这种直接传递一种情绪的战斗，那你写，就不用写太多文字了，直接一招解决就行了，因为你战斗的目的，仅仅是传递这一种情绪，而这种情绪，只需要解决对手就能办到，你写的冗长，毫无意义。

再比如，你救下的美女，其实是大家族的千金，你后面设置剧情，要去她家族，参加比一个比武。

那你写这场英雄救美的战斗，你的目的，其实要传递两种情绪，一种是美女对你的感激，另外一种，美女震惊你的天赋，因为震惊你的天赋，她才会邀请你去参加他们家族的比武。

所以，你这场战斗有传递两个情绪的目的，就必须展现天赋，就不能单单一招击败恶徒，要把那个恶徒写的修为比你高，你是越级将他击败的，这样你就传递两种情绪了，然后可以开战后面的剧情。

当然，还有传递三种情绪，前两种和之前一样，美女的感激，美女震惊你的天赋，甚至可以加一个，美女不仅仅震惊你的天赋，你施展的一招武技，竟然是他们家族失传的绝学，能修炼这门武技，就能开启他们家族的祖地。

好了，你设置了这个后续剧情。

那你写战斗，可以不仅仅是越级挑战了，甚至还可以将那门武技施展出来。

所以，一场战斗，想要写好，不是看打斗的精彩程度，而是看打斗的目的。

比如最精彩的生死决斗，为什么作者们都喜欢欲扬先抑，永远让对手一直占据上风，然后主角最后爆发最后的底牌击败对手？

生死决斗的目的，传递了很多情绪，一来是传递围观者情绪，他们各种惊叹主角和对手的实力，主角阵营的担忧，对手阵营的得意，轻蔑，二来，就是不断将各种不同的情绪，传递给对手，尤其是你铺垫了一个牛逼的底牌，最终肯定要用这个底牌击败对手，将这个底牌施展出去，也是一种情绪的爆发。

这一场战斗，各种不同复杂的情绪，传递出来，然后最终主角击败对手，主角将情绪彻底爆发出来，自然好看，爽！
