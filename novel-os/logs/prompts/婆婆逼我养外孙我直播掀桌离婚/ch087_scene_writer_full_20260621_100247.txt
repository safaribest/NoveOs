## 网文写作的核心特征（来自《斗罗大陆》等爆款玄幻）

1. 情绪可以直接写。斗罗大陆写"唐三很冷静""眼睛湿润了""发自内心的兴奋"——
   叙述者直接点明角色情绪，不是写动作暗示。读者不需要猜。
2. 对话标签用"道"为主（占80%），但允许"冷冷道""激动道""失声道"等情绪修饰。
   也允许用动作伴随对话："唐昊冷冷的看着他，道"。
3. 背景和设定可以大段交代。斗罗大陆开场就是"巴蜀，历来有天府之国的美誉"——
   直接交代世界背景，不需要全部嵌入场景。信息量大是网文的优势。
4. 精确数字可以使用。"十九下""十七道身影""两百年"——数字增加可信度。
5. 比喻可以用，但要是角色视角内的："如同雕像一样""宛如星丸跳跃"——不是公共库存比喻。
6. 段落短、切得碎。1-3句换段占70%。一句话一段很常见。阅读节奏要快。
7. 动作流程要拆细："以腿带腰，以腰带背，以背带臂"——每个环节都写出来。
8. 设定信息层层递进，不要一次倒完。从角色口中自然带出，不是旁白说明。
9. 每章要有"干货"——一个修炼细节、一个新设定、一个打斗技巧的拆解。
10. 【结构铁律】章节必须以钩子收尾——最后一个节拍是未解问题、新威胁或意外发现。没有钩子的章节视为失败。

# 开篇与代入感

## 开篇500字公式

```
1. 谁 + 在哪 + 正在经历什么 (100字)
2. 金手指/特殊之处是什么 (50字)
3. 眼前的冲突/危机 (200字)
4. 情感锚 — 读者为什么关心 (随叙事)
```

**番茄示范**: "小爷我乃镇南府世子林峰，不过是区区六百两酒钱，醉红楼我欠了三千两都不敢这么对我，还不快给小爷我松绑？"
→ 身份+地点+冲突+性格+悬念 全在一句话。

**错误**: 第一章大段背景交代、修炼体系说明、前世铺垫
**正确**: 从事件开始。设定通过角色的行动和对话自然带出

## 代入感六大支柱

1. **标签化** (~100字): 谁、长什么样、什么特征 → 读者脑中形成"可想象的空间"
2. **具体化**: 描写让读者不自觉地浮现出来的东西。读者会根据文字自动联想，不要帮读者联想
3. **可视化**: 用读者日常接触过的东西来类比。不可视的东西→读者无法代入
4. **情感锚定**: 开篇就给强烈情绪——愤怒、屈辱、困惑、兴奋——让读者立刻与之共情
5. **共鸣点**: 主角处境读者见过或能想象——被欺负、被误解、怀才不遇
6. **沉浸感**: 核心="主角就是我"。读者通过文字"经历"主角身上发生的事

## 开篇设计实战模板

**模板一: 危机开场**
```
主角正在面临一个具体的、迫在眉睫的危机
→ 展示他的处境和性格
→ 金手指/特殊能力介入
→ 化险为夷，但留下更大的悬念
```

**模板二: 身份反转开场**
```
主角表面身份被轻视/侮辱
→ 暗示/展示他的隐藏身份或能力
→ 冲突升级
→ 部分身份暴露，周围人震惊
→ 更大的身份谜团
```

**模板三: 重生/穿越开场**
```
前世最惨的时刻 → 回到关键节点 → 利用信息优势做第一个不同选择 → 产生蝴蝶效应
```
规则: 前世经历不要超过500字。重点是"今世我怎么做"。

## 进入故事的时机

- 凡人流: 从金手指获得开始（不是从童年开始）
- 战神文: 从回归/暴露身份的临界点开始
- 重生文: 从醒来的第一秒开始
- 系统流: 从系统激活的瞬间开始
- 规则: **永远从故事的倒数第二个节点开始，而不是第一个**

## 开头禁忌

- ❌ 大段背景介绍（世界设定、修炼体系、家族历史）
- ❌ 前世经历超过500字
- ❌ 主角在开篇"观望"而不是"行动"
- ❌ 引入超过3个有名字的角色
- ❌ 对话超过3轮没有推进剧情
- ✅ 第一个500字内: 主角做了一件具体的事

# 冲突、爽点与钩子

## 冲突单元公式

```
每章至少一个冲突单元:
欲望 + 障碍 = 冲突 → 行动 → 结果 → 新欲望

错误写法: 冲突→解决→平静→下一个冲突
正确写法: 旧冲突解决的同时，已经埋下新冲突的种子
```

|------|------|------|
| 打脸爽 | 被轻视→证明自己→对方震惊 | 通用 |
| 身份反转 | 以为是废物→其实是王者 | 都市战神/马甲文 |
| 信息差 | 我知道你不知道→利用碾压 | 重生/穿越 |
| 实力碾压 | 对手铺垫越强→赢得越爽 | 玄幻战斗 |
| 情感宣泄 | 受委屈→真相大白→道歉 | 言情/追妻 |

**关键规则**: 打脸不隔夜。一章内完成"被挑衅→反击→对方反应"。

## 钩子设计

**五类钩子**:

| 钩子 | 写法 | 位置 |
|------|------|------|
| 悬念钩子 | "他不知道的是……" / 刚出现的威胁 / 意外发现 | 章末 |
| 期待钩子 | 预告明天的大比 / 三天后的决战 | 章末 |
| 反转钩子 | 你以为是A→其实是B | 章中/末 |
| 身份钩子 | 隐藏身份即将被揭穿 | 章末 |
| 情感钩子 | 重要的人即将出现或离开 | 章末 |

**番茄拉新攻略**: 每500-800字一个信息落差。每章章末必须有钩子。章节标题要写"必看""身份揭露""碾压全场"这种期待感词。

## 节奏控制

- 战斗/打脸场景: 短句，快切，一句一段
- 过渡/日常场景: 可适当放慢但不超3段
- 爽点释放后: 立刻铺垫下一个期待，不要停
- "完读率"是番茄最重要的指标——高潮前置、不要拖、不要水

## 字数策略

番茄大神原话: "用事件数量控制字数，不是用描写长度。字数不够→加事件。字数超标→砍描写，不砍事件。"

一个完整事件链: 触发→行动→阻碍→意外→结果 ≈ 400-600字。每章3-5个事件自然落在2000-2500字。

# 新手速查：模板、骨架与避雷

## 开篇三步公式

```
第一步: 危机/冲突 → 主角立即行动（前500字必须）
第二步: 展示金手指/特殊之处（第一章内）
第三步: 设下长期期待（前3章内明确主线方向）
```

**反例**: "第一章: 主角醒来→想前世→看房间→见到丫鬟→出门散步" ← 读者已经跑了
**正例**: "主角睁眼发现自己在乱葬岗，断剑插在掌心，妖兽正在靠近" ← 前100字三大要素齐全

## 大纲骨架

**新手必备** (大神共识: 200章以上必须完整大纲):

```
卷1 (1-20章): 金手指激活+第一次逆袭+小高潮
卷2 (21-40章): 进入更大舞台+结识关键盟友+中期冲突
卷3 (41-60章): 遭遇重大挫折+突破瓶颈+反击准备
卷4 (61-80章): 正面决战+真相揭露+阶段性胜利
```

**章级骨架**:
```
每章 = 试炼/冲突(1-2个) + 成长收获 + 新悬念
```

## 六大新手雷区

| 雷区 | 表现 | 改正 |
|------|------|------|
| 开篇大段背景 | 第一章介绍世界观、修炼体系、家族历史 | 从事件开始，设定通过行动带出 |
| 前世写太长 | 重生文开场5000字写前世 | 前世≤500字 |
| 主角不行动 | 第一章主角在"看"和"想" | 主角做的第一件事就是第一章的核心事件 |
| 系统说明书 | 系统规则用500字逐条解释 | 系统在行动中展示——让读者看到它怎么运作 |
| 人物说话一个腔 | 丫鬟和皇帝说话风格一样 | 给每个角色一个口癖或话语特征 |
| 战斗解说员 | 打斗中插入招式说明和修炼原理解释 | 战斗只有动作和结果——体系在战斗外讲 |

## 新手自查清单

写完一章后自问:
- [ ] 这章有至少一个具体事件吗？（不是"思考""回忆""赶路"）
- [ ] 章末有钩子吗？（读者会想翻下一章吗？）
- [ ] 主角有主动做选择吗？（不是被推着走）
- [ ] 有新信息被揭示吗？（不是重复已知信息）
- [ ] 对话推动剧情了吗？（不是闲聊）
- [ ] 情绪变化有事件支撑吗？（不是突然开心/突然愤怒）
- [ ] 字数在目标范围内吗？（不是硬凑）

## 剧情平淡诊断

**如果感觉剧情平淡**: 不是描写不够，是冲突不够。
- 加障碍: 主角要做的这件事难度翻倍
- 加代价: 做这件事的后果更严重
- 加时间限制: 必须在XX之前完成

**番茄金句**: "读者说我写的平淡，永远不是在说你的文笔平淡——是在说你的冲突不够。"

## 从扑街到签约的路径

1. 选自己擅长+喜欢的题材（不要追热点）
2. 研究排行榜前20的同类型书（拆解他们的技巧）
3. 写完整大纲（新手必须）
4. 存稿5-8万字再发书
5. 前3章反复修改到最好状态
6. 保持日更4000+，不断更
7. 推荐前不要看数据——那个阶段的数据毫无意义
8. 第一本书大概率扑街——这是正常的。总结教训写下一本
